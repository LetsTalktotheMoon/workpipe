# Engineer - Services RF (Network Engineer)

**Company:** Ericsson
**Location:** Plano, TX
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Ericsson Inc. is seeking an Engineer – Services RF (Network Engineer) to join their team in Plano, TX. The role involves developing web applications, performing server installation and maintenance, and engaging with internal customers to gather requirements and define solutions.

## Core Skills
- Python programming
- JavaScript
- Shell scripting
- Perl programming
- Django

## Minimum qualifications
- Master's degree in Computer Science
- Computer Engineering, or a related field and 2 years of experience performing software development and automation
- 2 years of experience to include: Python programming
- Javascript
- Agile methodologies
- System design
- Server-side programming

## Responsibilities
- Perform access management, server installation, and service maintenance involving databases and web servers in Linux OS
- Automate server tasks and web programming using Python
- Shell
- JavaScript, and Perl programming
- Engage with internal customers and departments to gather and analyze solutions requirements, define solution milestones, and determine levels of effort
- Develop software solutions and modules to support RAN optimization tasks, such as: web apps interfaces, data parsers and automated excel reports, automated network data collectors, and radio nodes script generators
- Develop REST APIs for existing and new software solutions
- Commission, integrate, and configure 4G and 5G RAN Nodes
- Maintain and support Linux servers where applications are hosted
