# Software Engineer, Supply Chain Systems

**Company:** Google
**Location:** Sunnyvale, CA
**Seniority:** Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Google is a leading technology company that develops next-generation technologies to change how billions of users connect and interact with information. As a Software Engineer, you will design, develop, and maintain platforms to optimize Google's technical infrastructure and support supply chain operations.

## Core Skills
- Software Development
- Programming Languages - Java
- API Development - REST
- API Development - gRPC
- Database Systems - Spanner

## Minimum qualifications
- Bachelor's degree or equivalent practical experience
- 2 years of experience with software development in one or more programming languages, or 1 year of experience with an advanced degree
- Experience building and consuming APIs (e.g.
- REST, gRPC)
- Experience with database systems (e.g.
- Spanner
- BigQuery
- PostgreSQL)
- Experience deploying and managing applications on cloud infrastructure
- Experience in the Java programming language

## Preferred qualifications
- Master's degree or PhD in Computer Science or related technical fields
- 2 years of experience with data structures and algorithms
- Experience working on internal tools or enterprise systems and collaborating with internal stakeholders
- Experience in multiple full system implementation life-cycles (e.g., analyze, design, build, test, implement, support)
- Experience developing accessible technologies
- Familiarity with machine learning concepts and frameworks

## Responsibilities
- Design, develop, and maintain extensible platforms, leveraging technologies like Agentic AI, to optimize the design, planning, and deployment of Google's technical infrastructure
- Write, test, and deploy high-quality, efficient code for services supporting critical hardware and supply chain operations, taking ownership of feature development
- Collaborate with partner teams (e.g.
- Engineering
- Operations
- Product) to define requirements, influence solution design, and deliver technical solutions
- Troubleshoot, debug, and resolve complex issues in existing codebases, ensuring system reliability and performance
- Contribute to code reviews, documentation, and the continuous improvement of engineering best practices within the team
