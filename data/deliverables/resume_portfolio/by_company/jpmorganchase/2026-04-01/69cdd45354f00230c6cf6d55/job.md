# Software Engineer III - Full Stack Cloud (Java, React, AWS, GraphQL)

**Company:** JPMorganChase
**Location:** Palo Alto, CA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
JPMorgan Chase is one of the oldest financial institutions, offering innovative financial solutions. The Software Engineer III - Full Stack Cloud role involves designing and delivering technology products while executing critical technology solutions across various business functions.

## Core Skills
- Java
- Object-oriented programming
- TypeScript
- React.js
- NodeJS

## Minimum qualifications
- Formal training or certification on software engineering concepts and 3+ years applied experience
- Strong proficiency in Java and object-oriented programming
- Proficiency in front-end development using TypeScript
- React.js and NodeJS
- Experience with AWS services, specifically EKS (Elastic Kubernetes Service) and ECS (Elastic Container Service)
- Experience with infrastructure as code using Terraform
- Experience developing and consuming GraphQL APIs
- Familiarity with RESTful APIs, microservices architecture, and cloud-native development
- Experience with CI/CD tools and processes
- Excellent problem-solving and communication skills
- Overall knowledge of the Software Development Life Cycle

## Preferred qualifications
- Experience with other AWS services (e.g.
- Lambda
- S3
- RDS)
- Familiarity with containerization (Docker) and orchestration (Kubernetes)
- Knowledge of testing frameworks and tools (JUnit
- Jest, etc.)
- Familiarity with AI development tools such as GitHub Copilot and Codeium
- Experience with Agile methodologies
- Contributions to open-source projects or technical communities
- Experience working on platforms for external clients, especially in the commercial real estate or financial services domain

## Responsibilities
- Executes software solutions, design, development, and technical troubleshooting with ability to think beyond routine or conventional approaches to build solutions or break down technical problems
- Design, develop, and maintain full stack web applications using Java
- TypeScript
- React.js, and NodeJS
- Build and deploy cloud-native applications on AWS, with a focus on EKS (Elastic Kubernetes Service) or ECS (Elastic Container Service)
- Partner with UX designers to implement intuitive user interfaces that align with design systems
- Develop and maintain infrastructure as code using Terraform to ensure consistent, repeatable deployments and implement and optimize GraphQL APIs for efficient data access and integration
- Collaborate with cross-functional teams to define, design, and deliver new features for external clients
- Creates secure and high-quality production code and maintains algorithms that run synchronously with appropriate systems
- Implement CI/CD pipelines using Jenkins and Spinnaker for automated integration, testing, and deployment
- Ensure application security, scalability, and reliability in cloud environments
- Contributes to software engineering communities of practice and events that explore new and emerging technologies
