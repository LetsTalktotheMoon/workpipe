# Software Engineer 1

**Company:** ModMed
**Location:** Boca Raton, FL
**Seniority:** New Grad, Entry Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
ModMed is a company focused on modernizing the healthcare experience through innovative software solutions. They are seeking a highly motivated Software Engineer 1 to contribute to the development and enhancement of their healthcare technology, working under experienced engineers to deliver high-quality software that improves patient care.

## Core Skills
- Object-oriented programming
- Data structures
- AI coding assistants
- Large language models (LLMs)
- Code refactoring

## Minimum qualifications
- Bachelor's degree in Computer Science
- Software Engineering, or a related technical field
- 0-2 years of professional software development experience (internships, co-ops, or relevant academic projects will be considered)
- Solid understanding of object-oriented programming (OOP) principles and data structures
- Demonstrated experience using AI coding assistants (e.g.
- GitHub Copilot
- Cursor) to accelerate code generation, automate boilerplate reduction, and streamline debugging workflows
- Proven ability to leverage LLMs to refactor legacy codebases, improve code readability, and generate comprehensive documentation and unit tests
- Proficiency in at least one modern programming language (e.g.
- C#
- Java
- Python
- JavaScript)
- Basic understanding of software development lifecycle (SDLC) and agile methodologies
- Strong problem-solving skills and an analytical mindset
- Excellent written and verbal communication skills
- Ability to work effectively both independently and as part of a team

## Preferred qualifications
- Familiarity with web development concepts (e.g.
- HTML
- CSS
- JavaScript, front-end frameworks) and/or database technologies (e.g.
- SQL
- NoSQL) is a plus
- Prior experience with healthcare IT or a regulated industry is a bonus, but not required

## Responsibilities
- Collaborate with senior engineers and cross-functional teams (Product
- QA
- UX) to understand requirements and design software solutions
- Write clean, efficient, and well-documented code primarily in [mention specific languages, e.g.
- C#
- Python
- JavaScript] within an agile development environment
- Participate in code reviews, providing and receiving constructive feedback to ensure code quality and adherence to best practices
- Assist in identifying, diagnosing, and resolving software defects and performance issues
- Contribute to the testing of software components, including unit, integration, and system testing
- Learn and apply new technologies, tools, and methodologies relevant to software development and the healthcare industry
- Actively participate in team meetings, stand-ups, and other collaborative activities
- Adhere to company coding standards, security policies, and development processes
