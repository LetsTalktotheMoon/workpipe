# Software Engineer - Frontend

**Company:** Snowflake
**Location:** Bellevue, WA
**Seniority:** Entry, Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Snowflake is a leading company in the data revolution, focused on building an effective and accessible cloud data platform. They are looking for a Frontend Software Engineer to design and develop interactive user interfaces for their AI Data Cloud, collaborating closely with designers and product managers to create exceptional user experiences.

## Core Skills
- Frontend development
- React
- TypeScript
- Computer science fundamentals
- Algorithms

## Minimum qualifications
- 2–5 years' of industry experience specializing in frontend development for scalable web applications
- Strong foundation on the modern frontend stack, including fluency React and Typescript
- Strong fundamental computer science skills
- Knowledge of algorithms and data structures
- A strong understanding of core computer science concepts and web application architecture, including security, performance, and scalability
- A genuine passion for building customer-facing products and a drive to solve problems that create real value for users
- Experience collaborating with backend teams to define API contracts that power dynamic frontend experiences
- BS/MS/PhD in Computer Science or related majors, or equivalent experience

## Responsibilities
- Design, develop, and deliver highly interactive and responsive user interfaces that have a direct impact on our customers and partners
- Work closely with Designers and Product Managers to translate complex requirements into intuitive workflows
- Architect and build robust, reusable frontend components and state management systems using TypeScript and React
- Optimize web performance to ensure a seamless experience, focusing on core web vitals and efficient data rendering
- Write clean, well-tested, and maintainable code, while contributing to our shared design system and internal frontend standards
