# Software Testing Engineer

**Company:** Centific
**Location:** Redmond, Washington
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Centific is a frontier AI data foundry that curates diverse, high-quality data to empower clients with scalable AI deployment. They are seeking a Software Testing Engineer to validate application functionality, design and execute test plans, and collaborate with development teams to ensure high-quality software releases.

## Core Skills
- Software Testing
- Quality Assurance
- Compatibility Engineering
- Hardware Validation
- Software Validation

## Minimum qualifications
- BS degree in CS or equivalence
- Previous experience in QA, software testing, or compatibility engineering, often requiring 3–5+ years in the industry
- Strong understanding of hardware/software validation, test methodologies, and bug tracking tools
- Ability to analyze failures and identify root causes in complex systems
- Excellent written and verbal communication skills to report bugs and document test results

## Responsibilities
- Validate application functionality across different operating systems
- Design, develop, and execute test plans, test cases, and test protocols to identify defects, layout scaling issues, and crashes
- Document software defects in bug tracking systems, perform initial debugging by analyzing logs, and collaborate with developers to find root causes
- Install and configure simulated or real production environments to test software performance
- Work with development teams, product managers, to review requirements and ensure high-quality releases
