# Full Stack Cloud Engineer (AI Security Platform)

**Company:** HiddenLayer
**Location:** United States
**Seniority:** Mid Level
**Work Model:** Remote
**Employment Type:** Full-time

## About the job
HiddenLayer is a company dedicated to protecting valuable technologies from adversarial AI attacks. They are seeking a Full Stack Cloud Engineer to design and implement microservices, build cloud-native services, and enhance customer-facing product experiences while ensuring performance and operational excellence.

## Core Skills
- Go
- Python
- TypeScript
- React
- AWS

## Minimum qualifications
- Strong full-stack experience shipping and operating production software
- Proficiency with Go and/or Python (service development in production)
- Proficiency with TypeScript + React
- Hands-on experience with AWS or Azure
- Strong experience with Kubernetes + Docker / containers
- Experience designing or working in microservices / distributed systems, ideally with low-latency requirements
- Experience in Agile delivery environments (iterative planning, continuous improvement)
- Track record of building automated test coverage that supports safe continuous deployment
- Strong communication skills
- you proactively align stakeholders and surface tradeoffs
- Comfort with ambiguity and a bias for action, you move forward while clarifying what needs to be figured out

## Preferred qualifications
- Exposure to ML/AI concepts or building adjacent to ML workflows
- Experience building security products, cloud security tooling, or security ecosystem integrations
- Familiarity with observability (OpenTelemetry, metrics/logs/traces) and defining SLOs/SLIs
- Experience with API gateways, service mesh, event-driven systems, or contract testing
- Infrastructure-as-Code experience (Terraform
- Pulumi
- CloudFormation
- Bicep)

## Responsibilities
- Ship end-to-end platform features (UI → API → cloud services) that customers rely on daily
- Improve performance and reliability of low-latency, distributed services running in Kubernetes
- Increase deployment confidence and velocity through strong automated test coverage and CI/CD
- Deliver integrations that connect HiddenLayer into the broader cloud
- AI, and security ecosystems
- Design and implement microservices in Go and/or Python, with clear API contracts and interoperability
- Build and operate cloud-native services on AWS or Azure, deployed as containerized workloads on Kubernetes
- Build and evolve customer-facing product experiences using TypeScript + React
- Own performance: profile, troubleshoot, and optimize p95/p99 latency, throughput, and resource efficiency
- Strengthen delivery: maintain test strategies (unit/integration/e2e) that enable continuous deployment
- Improve operational excellence: logging/metrics/tracing, alerting, on-call readiness, runbooks, and incident follow-ups
