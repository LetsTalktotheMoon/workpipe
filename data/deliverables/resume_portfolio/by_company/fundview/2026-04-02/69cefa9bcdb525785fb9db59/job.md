# Software Developer (Entry Level)

**Company:** FundView
**Location:** Lubbock, TX
**Seniority:** New Grad, Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
FundView is a company that builds software to help local governments operate clearly and responsibly. They are seeking a detail-oriented Entry-Level Software Developer to assist in designing, developing, testing, and maintaining web applications while collaborating with a team to improve functionality and user experience.

## Core Skills
- Object-Oriented Programming
- HTML
- CSS
- SQL
- Eagerness to learn

## Minimum qualifications
- Bachelor's degree in Computer Science, a related field, or equivalent experience
- Foundational understanding of Object-Oriented Programming concepts
- Foundational understanding of HTML/CSS
- Foundational understanding of SQL and basic database principles
- Strong attention to detail and eagerness to learn
- Ability to work collaboratively in a fast-paced, in-office environment

## Responsibilities
- Assist in designing, developing, testing, and maintaining web applications
- Write clean, maintainable code based on project requirements
- Identify, troubleshoot, and resolve software bugs and issues
- Test and validate code to ensure quality and performance standards are met
- Collaborate with team members to continuously improve functionality and user experience
