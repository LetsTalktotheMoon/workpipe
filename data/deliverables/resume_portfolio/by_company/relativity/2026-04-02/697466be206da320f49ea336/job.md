# Advanced Software Engineer

**Company:** Relativity
**Location:** Illinois, United States
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Relativity is a leading legal data intelligence company that provides technology for organizing data and uncovering truth in high-stakes legal matters. The Advanced Software Engineer will design, develop, and operate secure, scalable, cloud-native web applications while collaborating with cross-functional teams to solve complex data challenges.

## Core Skills
- Distributed Systems
- Cloud-native Systems
- C#
- Java
- Python

## Minimum qualifications
- Bachelor's degree in Computer Science
- Engineering, or a related field, or equivalent practical experience
- 3+ years of professional software engineering experience, with a focus on distributed or cloud-native systems
- Proficiency in at least one modern programming language such as C#
- Java, or Python
- Experience building and operating cloud-based services using platforms such as Azure or AWS
- Strong understanding of software design principles, performance optimization, and secure coding practices
- Ability to write maintainable, well-tested code and work effectively in a collaborative team environment
- Automation
- Data Analysis
- Database Management
- Network Architecture

## Preferred qualifications
- Experience with containerization technologies and modern CI/CD or DevOps pipelines
- Familiarity with observability, reliability engineering, or performance monitoring tools
- Experience working on large-scale data processing or AI-enabled systems

## Responsibilities
- Design, develop, and operate secure, scalable, cloud-native web applications using modern architecture and object-oriented principles
- Build and operate AI-driven, high-performance systems that process and analyze large volumes of data
- Deliver clean, testable, maintainable code with a strong focus on quality, performance, and reliability
- Collaborate closely with engineers and product managers through pair programming, design sessions, and code reviews
- Actively contribute to design and code reviews, providing thoughtful feedback and sharing technical knowledge
- Participate in service ownership, including on-call rotations and troubleshooting production issues in distributed systems
- Champion engineering best practices including automated testing, static analysis, observability, and DevOps workflows
- Leverage AI-assisted development tools such as GitHub Copilot or Cursor to improve productivity and focus on business-critical problems
