# Software Developer

**Company:** FlexTrade
**Location:** Great Neck, NY
**Seniority:** Entry, Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
FlexTrade Systems is a global leader in high performance multi-asset execution management and order management systems. They are seeking a skilled Backend Developer with expertise in Golang and AWS services to design, develop, and maintain scalable backend systems while ensuring high performance and reliability.

## Core Skills
- Golang
- AWS services
- Docker
- Linux operating systems
- Shell scripting

## Minimum qualifications
- 2–4 years of experience in backend development
- Strong proficiency in Golang
- Experienced in Docker and containerized deployments
- Hands-on experience with AWS services
- Solid understanding of Linux operating systems and shell scripting
- Experience with Redis for caching and data management
- Good understanding of API design, microservices, and distributed systems
- Experience with SQL/NoSQL databases
- Knowledge of CI/CD pipelines and version control (Git)
- Bachelor's degree in computer science
- Engineering, or related field (or equivalent experience)

## Preferred qualifications
- Experience with React or some front-end technologies
- Experience with Kubernetes or other orchestration tools
- Familiarity with serverless architecture
- Knowledge of monitoring and logging tools (CloudWatch
- Prometheus, etc.)

## Responsibilities
- Design, develop, and maintain backend services using Golang
- Build and optimize RESTful APIs and microservices architecture
- Implement and manage cloud infrastructure using AWS services
- Ensure application performance, scalability, and security
- Collaborate with cross-functional teams to define and deliver new features
- Write clean, maintainable, and testable code following best practices
- Participate in code reviews and contribute to continuous improvement
- Integrate and optimize Redis for caching and data storage
