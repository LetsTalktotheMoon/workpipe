# Software Engineer II, Backend Full Stack - Financial Product

**Company:** Rippling
**Location:** San Francisco, CA
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Rippling is a company that integrates HR, IT, and Finance systems for businesses. They are seeking a Software Engineer II to design and implement features with a focus on backend logic and API development, while also contributing to frontend implementation across their financial product offerings.

## Core Skills
- Full-stack development
- Backend development
- API development
- Python
- Go

## Minimum qualifications
- 2+ years of professional experience with a focus on full-stack development, demonstrating the ability to deliver reliable, end-to-end products using a backend language (e.g.
- Python/Go) and frontend technologies like React/TypeScript
- Prioritize projects based on business impact, ensuring high-quality implementation without unnecessary complexity
- Ability to thrive in dynamic environments, quickly switching context between frontend and backend work as needed to accelerate delivery
- Excellent skills in communicating technical trade-offs across the stack with clarity and precision to both frontend and backend partners

## Responsibilities
- Design and implement features with an emphasis on backend logic
- API development, and data integrity, while also contributing to the frontend implementation
- Work across the stack, focusing on API design, event modeling, and robust data management while ensuring a seamless connection to the user interface
- Own the full development lifecycle for features, ensuring that the backend architecture supports both reliable functionality and excellent frontend performance
- Work closely with cross-functional partners to understand requirements, refine technical scope, and drive successful, reliable feature delivery
