# Software Engineer, Server

**Company:** Lyft
**Location:** San Francisco County, CA
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Lyft is looking for product-focused server software engineers for our Rider Team, aiming to build technologies that enrich the lives of the community. As an engineer, you will collaborate with various teams to enhance the user-facing app experience for requesting and completing rides.

## Core Skills
- Python
- Go
- Object-oriented programming
- Data structures
- Algorithms

## Minimum qualifications
- BS/MS or equivalent in Computer Engineering
- Computer Science, or related field or equivalent practical experience, +3 years of experience working
- Experience with Python and/or Go, or a similar language
- Proficiency in object-oriented programming
- Experience working with data structures or algorithms
- Ability to work with a low-ego, highly collaborative, and cross-functional team

## Preferred qualifications
- Experience pursuing side projects or open-source projects

## Responsibilities
- Write well-crafted, well-tested, readable, maintainable code
- Participate in code reviews to ensure code quality and distribute knowledge
- Participate in our team's on call rotation
- Identify, triage, debug and resolve issues/bugs across our various applications and platforms
- Have the ability to explain the various trade offs made in decisions
- Manage project priorities, deadlines, and deliverables
