# Software Development Engineer 3

**Company:** WEX
**Location:** United States
**Seniority:** Mid, Senior Level
**Work Model:** Remote
**Employment Type:** Full-time

## About the job
WEX is a company specializing in modern, cloud-native environments, and they are seeking a Software Development Engineer 3 to bridge the gap between high-scale software development and advanced system observability. The role focuses on implementing the 'Observability-as-Code' strategy, managing incident response, and creating documentation to empower future engineers.

## Core Skills
- Software Development
- Python
- Java
- Go
- Software Development Life Cycle

## Minimum qualifications
- Experience in software development (Python
- Java
- Go, or similar) with a strong grasp of the full SDLC
- 5+ years of proven experience with Grafana Cloud and Splunk Cloud
- Comfortable managing Grafana Cloud and Splunk Cloud via code (Terraform providers or dedicated CLI tools) rather than just manual UI configuration
- Practical experience with cloud-based tools and internet-related infrastructure

## Preferred qualifications
- Experience configuring xMatters for intelligent automated alerting and on-call workflows
- Experience with Axonius for cybersecurity asset management and identifying coverage gaps in security tooling
- Strong understanding of CI/CD pipelines and the ability to integrate observability metrics into deployment gates

## Responsibilities
- Utilize Grafana Cloud and Splunk Cloud to build deep visibility into system health
- Manage these platforms using Configuration-as-Code (e.g.
- Terraform
- Grafana Grizzly, or Splunk Monitoring-as-Code) to ensure environment parity and version-controlled dashboards
- Integrate xMatters for automated incident routing and communication
- Utilize Axonius to maintain a comprehensive, real-time inventory of cyber assets and ensure security compliance across the tech stack
- Create and maintain high-fidelity technical documentation and runbooks to empower future engineers and managers to resolve issues independently
