# Senior Software Engineer

**Company:** SharkNinja
**Location:** Needham, MA
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
SharkNinja is a global product design and technology company known for its innovative lifestyle solutions. They are seeking a Senior Software Engineer to enhance their Sales and Demand Planning application by developing RESTful APIs and optimizing data strategies.

## Core Skills
- JavaScript
- Node.js
- AngularJS
- Vue.js
- SQL Server

## Minimum qualifications
- 5 to 7 years of Internet development experience
- 5+ years of SQL query and stored procedure development experience
- Deep understanding of modern single-page MVC frameworks
- Experience integrating multiple data sources and databases into one system
- Proficient knowledge of a back-end programming language, especially JavaScript
- In depth knowledge of SQL Server and experience across versions 2014
- 2019 and Azure
- Proficient understanding of GitHub
- Expert level JavaScript programming, including Node and AngularJS
- Significant SQL development and modeling
- Own tasks to completion in a detailed-oriented, methodical manner

## Preferred qualifications
- Bachelor's degree in Information Systems
- Computer Science, or equivalent experience is preferred
- In depth knowledge of the SDLC, particularly Agile methodologies
- Experience programming with BI tools
- Experience with Python and RabbitMQ
- Experience with MongoDB
- Knowledge of wholesale distribution, retail, or manufacturing industry is also a plus

## Responsibilities
- RESTful API development
- Asynchronous node.js coding
- AngularJS and Vue front-end development
- Build reusable code and libraries for future use
- Optimize application for maximum performance and scalability
- Implement security and data protection
- Design and implement data storage solutions
- Collaborate with team in solving complex problems
- Able to work independently on assigned tasks
- Follow, enhance, and contribute to group coding standards and best practices around documentation, code-formatting, source-control
