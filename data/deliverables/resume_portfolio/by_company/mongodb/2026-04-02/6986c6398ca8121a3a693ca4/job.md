# Software Engineer 3, Atlas Data Federation Customer Experience

**Company:** MongoDB
**Location:** New York City

**Work Model:** Hybrid

## About the job
MongoDB is a leading technology company that empowers customers to innovate with its unified database platform. The role of Software Engineer 3 involves designing, building, and testing high-impact projects to enhance the Atlas Data Federation system's scalability and performance, while collaborating with teams to deliver new features.

## Core Skills
- Go
- Java
- React
- Distributed storage systems
- Cloud-native backend systems

## Minimum qualifications
- 3+ years experience in software engineering, primarily focused on cloud-native backend systems
- B.Sc / M.Sc in Computer Science or similar field, or equivalent practical experience
- Proficient in a modern compiled programming language (Go
- Java
- C#
- C++, etc.)
- Experience implementing, and operating distributed storage systems with high availability and durability guarantees
- Enjoys working with others towards a common goal
- You're a self-starter! You're comfortable organizing your own time and propose changes backed by data
- Is curious, collaborative, empathetic, coachable and intellectually honest

- Has a passion for problem solving and learning new things in the domains of computer science and software engineering

- Experience building and testing full-stack systems (using React or similar frameworks)

- Experience designing Data Lake protocols (and similar e.g. Delta Lake

- Experience optimizing throughput in distributed storage systems
- Experience working with multiple cloud providers (AWS
- GCP

- Design, architect, build, and deliver core pieces of Atlas with focus on storage systems, query engine, and customer facing components with coding in Go
