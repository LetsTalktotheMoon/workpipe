Bloom Energy is a company focused on clean, reliable, and affordable energy solutions. They are seeking an MES Software Engineer to develop software solutions, support and maintain software functionality, and ensure security protocols are implemented in a manufacturing environment.

- Python
- API Development

- Experience with modern programming languages like Java
- Python, etc

- Deep understanding of SQL and experience with systems like MySQL

- MongoDB, etc
- Support, maintain and document software functionality
