# Senior Software Developer - .NET

**Company:** Hone Health
**Location:** United States
**Seniority:** Mid, Senior Level
**Work Model:** Remote
**Employment Type:** Full-time

## About the job
Hone Health is an online medical clinic focused on transforming healthcare and enhancing longevity. They are seeking a Senior Software Developer to develop, enhance, and maintain core applications built with C#/.NET, ensuring high performance, scalability, and reliability while mentoring other developers.

## Core Skills
- C#/.NET Core
- SQL Server
- Azure SQL
- API design, development

## Minimum qualifications
- At least 5 years production experience with C# / .NET (Core) and SQL Server (or Azure SQL)
- Hands-on API design and development experience and proficiency in API frameworks/tools such as Swagger and Postman
- Mastery of development tooling (Visual Studio
- Git, etc)
- Comfortable with Entity Framework and Razor pages
- Demonstrated experience integrating AI tools and services (e.g.
- GitHub Copilot
- ChatGPT
- AI-based code analyzers) into daily development workflows to boost productivity, automate repetitive tasks, and support decision-making
- At least 3 years experience with cloud platforms (Azure
- AWS, or GCP)
- Familiarity with front end frameworks like React
- Angular, or Vue
- Firm grasp of lifecycle best practices, including source control, testing (unit, performance, automation), security, maintainability, code reuse, etc
- Work well both independently and in a team/paired programming setting

## Responsibilities
- Develop high-quality, scalable software solutions that drive business growth and operational efficiency
- Design end-to-end technical solutions in collaboration with business leaders, ensuring alignment with strategic goals
- Deliver robust, maintainable products by applying sound design principles, best practices, and rigorous quality standards
- Work cross-functionally with teams across the organization to solve complex technical and business challenges
- Mentor and guide developers, fostering a culture of continuous improvement and helping elevate team performance
