# Java Full Stack Developer

**Company:** Deloitte
**Location:** Dallas, TX
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Deloitte is a leading consulting firm seeking an experienced Java Full Stack Developer - Project Delivery Senior Analyst. The role involves designing and building full-stack features, ensuring delivery excellence, and leading client engagement workstreams focused on process improvement and operational outcomes.

## Core Skills
- Java Full Stack Development
- REST API Development
- Microservices Architecture
- API Design
- Service-to-Service Authentication

## Minimum qualifications
- 3+ years of experience in software, systems, or embedded engineering
- 3+ years of experience working Java Full Stack
- 3+ years of experience creating REST APIs
- Proven delivery of microservices (REST/async messaging)
- API design, and service-to-service auth
- Strong hands-on experience with Kubernetes, containerization, and GitOps (e.g.
- ArgoCD/Flux) operating production systems
- Understanding of network protocols, security best practices, and scalable API design
- Experience with Git
- CI/CD pipelines, and modern DevOps practices
- Bachelor's degree, preferably in Computer Science
- Information Technology
- Computer Engineering, or related IT discipline
- or equivalent experience
- Limited immigration sponsorship may be available
- Ability to travel 10%, on average, based on the work you do and the clients and industries/sectors you serve

## Preferred qualifications
- Proven ability to communicate complex technical concepts clearly and effectively
- Analytical ability to manage multiple projects and prioritize tasks into manageable work products
- Can operate independently or with minimum supervision
- Excellent Written and Communication Skills
- Ability to deliver technical demonstrations

## Responsibilities
- Rationalize PCS service providers into a select set of strategic partners across Engineering and Cloud Security Services
- Drive measurable value through cost optimization, improved productivity, enhanced customer experience, and simplified operations
- Ensure delivery excellence with a focus on quality, speed, and innovation
- Design and build full-stack features: React UI + Java/Spring Boot (and/or Python) backend services with clean APIs and robust error handling
- Own microservice architecture concerns: service boundaries, contracts, versioning, backwards compatibility, and resilience patterns
- Implement and maintain CI/CD and GitOps workflows
- improve developer productivity and deployment reliability
- Deliver experimentation capabilities: integrate feature flags, run A/B tests, analyze outcomes, and iterate safely
- Engineer for scale: address consistency issues, race conditions, concurrency controls, and distributed transaction alternatives (sagas/outbox)
- Communicate regularly with Engagement Managers (Directors), project team members, and representatives from various functional and / or technical teams, including escalating any matters that require additional attention and consideration from engagement management
- Independently and collaboratively lead client engagement workstreams focused on improvement, optimization, and transformation of processes including implementing leading practice workflows, addressing deficits in quality, and driving operational outcomes
