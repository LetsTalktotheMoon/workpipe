# Software Development Engineer, OneMHS

**Company:** Amazon
**Location:** Arlington, VA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Amazon is building the next generation software, hardware, and processes to run their global fulfillment network. They are seeking a software development engineer to integrate robotics within their advanced warehouses, focusing on automation projects and edge compute solutions for industrial applications.

## Core Skills
- Software Development Life Cycle
- Software Programming
- Object-Oriented Design
- System Architecture
- Design Patterns

## Minimum qualifications
- 3+ years of non-internship design or architecture (design patterns, reliability and scaling) of new and existing systems experience
- 3+ years of non-internship professional software development experience
- 3+ years of full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations experience
- Experience programming with at least one software programming language

## Preferred qualifications
- 3+ years of programming using a modern programming language such as Java
- C++, or C#, including object-oriented design experience
- 2+ years of contributing to new and current systems architecture and design (architecture, design patterns, reliability and scaling) experience
- Experience with AWS technologies
- Travel up to 20% of the time
- 3+ years of experience developing SaaS and IaaS applications utilizing AWS CDK in TypeScript

## Responsibilities
- Integrate with a diverse array of internally and externally developed hardware and software services
- Implement continuous integration and continuous deployment (CI/CD) pipelines
- Validate solutions via software-in-the-loop simulation
- Work closely with deployment teams to implement end-to-end solutions in the field
- Develop infrastructure to support development, operations, and maintenance of low level software for warehouse automation systems
