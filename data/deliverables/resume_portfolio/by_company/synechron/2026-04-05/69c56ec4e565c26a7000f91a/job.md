# Python Fullstack Engineer

**Company:** Synechron
**Location:** Charlotte, NC
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Synechron is a global consulting firm that leverages digital solutions to transform businesses. They are seeking a talented and experienced Python Fullstack Engineer to design, develop, and maintain end-to-end web applications, ensuring seamless integration between frontend and backend components.

## Core Skills
- Python
- Java SpringBoot
- React.js
- Relational databases
- NoSQL databases

## Minimum qualifications
- Strong proficiency in Python for backend development
- Experience with Java SpringBoot framework
- Frontend development skills with React.js
- Experience with relational and NoSQL databases
- Hands-on experience with microservices architecture
- Knowledge of CI/CD tools like Jenkins
- GitLab CI, or CircleCI
- Strong understanding of RESTful API design and implementation
- Basic knowledge of cloud platforms (AWS
- Azure, or GCP)
- Familiarity with containerization and orchestration tools like Docker and Kubernetes
- Good understanding of agile methodologies and version control (Git)

## Preferred qualifications
- Experience with AI/ML integration
- NLP, or large language models
- Knowledge of SQL translation and agent development
- Understanding of modern frontend build tools and state management
- Experience with testing frameworks and automation

## Responsibilities
- Develop and maintain robust web applications using Python
- React.js, and Java SpringBoot
- Design and implement RESTful APIs to support frontend components and third-party integrations
- Build and optimize databases, ensuring data integrity and performance
- Architect and develop microservices-based solutions for scalability and flexibility
- Collaborate with product managers
- UI/UX designers, and other developers to deliver high-quality features
- Implement CI/CD pipelines to automate testing, deployment, and monitoring
- Participate in code reviews, troubleshoot issues, and continuously improve code quality
- Maintain documentation for system architecture
- APIs, and processes
- Stay updated with emerging technologies and industry best practices
