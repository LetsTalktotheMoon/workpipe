# Software Engineering

**Company:** Microsoft
**Location:** Redmond, WA
**Seniority:** Mid, Senior Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Microsoft is a leading technology company focused on empowering every person and organization to achieve more. They are seeking a skilled Developer to lead the design, development, and scaling of AI-powered multi-agent systems, ensuring integration with Microsoft 365 and driving innovation in agentic architecture.

- C++
- C#
- Java
- JavaScript

## Minimum qualifications
- Bachelor's Degree in Computer Science or related technical field AND 2+ years technical engineering experience with coding in languages including, but not limited to
- C
- C++
- C#
- Java
- JavaScript, or Python OR equivalent experience

## Preferred qualifications
- Master's Degree in Computer Science or related technical field AND 3+ years technical engineering experience with coding in languages including, but not limited to
- C
- C++
- C#
- Java
- JavaScript, or Python OR Bachelor's Degree in Computer Science or related technical field AND 5+ years technical engineering experience with coding in languages including, but not limited to
- C
- C++
- C#
- Java
- JavaScript, or Python OR equivalent experience
- Proficiency in Python
- C#, and TypeScript
- Experience with AI frameworks such as LangChain
- Semantic Kernel, and orchestration tools like Autogen
- Familiarity with LLMs (Large Language Models), vector databases, and RAG (Retrieval-Augmented Generation) pipelines
- Experience, ability, and willingness to work collaboratively and independently with others on the implementation of large, distributed systems
- Solid grasp of security, compliance, and data governance in enterprise environments
- Familiarity with Microsoft 365 extensibility
- Copilot Studio, and Microsoft Graph API
- Knowledge of MCP (Model Context Protocol) and agent coordination protocols
- Contributions to open-source AI or agentic frameworks

## Responsibilities
- Architect and implement scalable, secure, and compliant multi-agent systems using LLMs, orchestration frameworks, and Microsoft’s AI stack
- Integrate agents with Microsoft Graph
- Power Platform, and internal APIs to enable cross-system intelligence and automation
- Define and enforce architectural standards for agent interoperability, real-time processing, and fault tolerance
- Drive performance optimization and observability improvements across agentic workflows
- Oversee Live Site operations for agentic systems, ensuring reliability, rapid incident response, and continuous improvement
- Ensure compliance with Microsoft’s Responsible AI and security standards, including Role Based Access Control (RBAC), data masking, and audit logging
- Collaborate with partner engineering teams to build extensible frameworks that support third-party agent development and deployment
