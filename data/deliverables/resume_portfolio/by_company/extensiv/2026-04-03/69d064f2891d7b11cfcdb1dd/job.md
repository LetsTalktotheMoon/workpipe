# Software Engineer

**Company:** Extensiv
**Location:** Sandy, UT
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Extensiv is a visionary technology leader focused on creating the future of omnichannel fulfillment. They are seeking a Software Engineer to help build the next generation of their cloud-based fulfillment technology, working on scalable SaaS applications for modern eCommerce.

## Core Skills
- PHP
- Relational databases
- SQL
- SaaS application deployment, operation

## Minimum qualifications
- Degree in Computer Science
- Information Systems, or a related field
- 3 – 6 years of professional software development experience
- Proficiency with PHP (or a similar interpreted language) and relational databases (SQL/RDBMS)
- Experience deploying and operating SaaS applications
- Experience building RESTful Web Services and integrating APIs
- Familiarity with CI/CD pipelines
- Git (including hosted providers like GitLab), and Jira (or other issue-tracking systems)
- Solid understanding of software design principles and patterns
- Effective communication skills and the ability to work collaboratively in a cross-functional team
- Passion for learning, continuous improvement, and delivering quality software

## Preferred qualifications
- Experience with AWS
- Exposure to automated testing frameworks (Unit
- Integration, and/or End-to-End)
- Experience with performance optimization and monitoring tools

## Responsibilities
- Design, develop, and maintain cloud-based software solutions in an agile environment
- Write clean, maintainable, and testable code following industry best practices
- Participate in architecture and design discussions for new applications and features
- Collaborate with Product Management to review requirements and contribute to estimation and planning
- Build and integrate APIs and modern front-end components using technologies such as JavaScript and PHP
- Contribute to continuous integration and deployment pipelines (CI/CD) for timely, consistent delivery
- Write and maintain automated tests, including unit and integration testing
- Monitor application health and support production issues in collaboration with other teams
- Contribute ideas for technical improvements and process efficiency across the SDLC
