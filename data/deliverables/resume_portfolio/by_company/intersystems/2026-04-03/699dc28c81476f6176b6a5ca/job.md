# Applications Developer

**Company:** InterSystems
**Location:** Boston, MA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
InterSystems is a creative data technology provider that delivers a unified foundation for next-generation applications across various industries. The Applications Developer will design and develop applications for internal staff and customers, working closely with users to capture needs and enhance current applications.

## Core Skills
- Operating Systems - Windows
- Operating Systems - Unix
- User Interface Implementation
- XML
- XSL

## Minimum qualifications
- Experience in developing for multiple operating system environments including both Windows and Unix
- Experience with a variety of user interface implementation approaches
- Demonstrable competence in XML
- XSL
- SOAP
- Web Services, eCommerce
- HTML
- JavaScript
- Demonstrate competence in designing modern web applications based on REST and Angular
- Experience with Object Oriented design and development
- Familiarity with InterSystems IRIS
- Caché, or Ensemble
- Have contributed substantially to the deployment of an application-rich intranet or extranet

## Preferred qualifications
- Bachelor's degree or foreign equivalent in a technical field or STEM
- 3+ years of experience in a full stack development role
- Process improvement and/or program and project management background is a plus, preferably in a software or other technology company

## Responsibilities
- Work with internal users and capture needs for creating new applications as well as extending and enhancing current applications
- Create and deploy new functionality and applications for use by InterSystems staff and their customers by exploiting the full power of the company’s products
- Document the code and generate appropriate user documentation
- Work closely with product developers to apply newly introduced InterSystems IRIS features to new and existing applications
- Provide continuous feedback to the product development group regarding the new features in the product
- Provide technical customer support and consultation service related to product installation and application
- Assist in integrating company’s various applications
- Assume additional responsibilities as requested
- Manage projects
- initiate and drive multiple workflows to successful completion with strong communication and collaboration
- Support Integrations with legacy systems and other internal Applications
