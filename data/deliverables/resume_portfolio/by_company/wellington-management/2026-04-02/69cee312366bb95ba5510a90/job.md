# Software Engineer

**Company:** Wellington Management
**Location:** Boston, MA
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Wellington Management is a private partnership offering investment management capabilities across global capital markets. They are seeking a Software Engineer to join the Prism engineering team, focusing on core platform and infrastructure development for their enterprise data federation platform.

## Core Skills
- Java
- Spring Boot
- AWS
- Distributed Systems
- APIs

## Minimum qualifications
- Strong experience with Java-based backend development
- Experience with Spring Boot and AWS
- Solid understanding of distributed systems
- APIs, and service-oriented architectures
- Experience working on production systems with high availability and reliability requirements
- Familiarity with modern CI/CD pipelines, automated testing, and operational tooling
- Ability to diagnose and resolve complex production issues
- Strong communication skills and the ability to work effectively with onshore and offshore teams

## Preferred qualifications
- Experience with enterprise data platforms, data access layers, or federated data systems
- Experience with NodeJS and SQL
- Exposure to cloud-based infrastructure and platform services
- Experience applying AI or automation to software engineering workflows (e.g., testing, code quality, operational tooling)
- Background supporting large-scale, business-critical applications

## Responsibilities
- Design, develop, and support core components of the Prism platform, primarily in a Java
- Spring Boot
- AWS based environment
- Own and evolve critical infrastructure services that support firmwide data access
- Partner closely with offshore developers to provide technical direction, code reviews, and production support coverage
- Improve platform resiliency, scalability, and performance for high-volume, low-latency workloads
- Expand AI-driven automation across development, testing, and operational workflows
- Participate in incident response and root cause analysis for availability or performance issues
- Contribute to long-term platform modernization and reduction of operational risk
- Collaborate with application teams that consume Prism to ensure reliable and efficient data access patterns
