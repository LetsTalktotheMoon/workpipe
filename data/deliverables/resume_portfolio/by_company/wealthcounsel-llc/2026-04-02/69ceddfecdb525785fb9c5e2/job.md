# Junior Software Developer (Virtual, Full Time)

**Company:** WealthCounsel, LLC
**Location:** United States
**Seniority:** Entry Level
**Work Model:** Remote
**Employment Type:** Full-time

## About the job
WealthCounsel is committed to fostering a positive work environment and is seeking a motivated Junior Software Developer to join their team. In this role, you will collaborate closely with experienced engineers to design, develop, and maintain high-quality web applications that meet client needs.

## Core Skills
- TypeScript
- JavaScript
- Web application development
- Programming logic
- Debugging

## Minimum qualifications
- Bachelor's degree in Computer Science or a related field (or equivalent professional experience)
- 1-2 years of experience building scalable web applications with a deep focus on TypeScript/Javascript and core web standards
- Solid understanding of programming logic, debugging, and client-server architecture
- Familiarity with REST APIs and the principles of CI/CD
- A proven aptitude for learning new technologies and a proactive approach to mastering new programming languages

## Preferred qualifications
- Familiarity with Vue.js (or similar frameworks like React or Angular) and state management patterns
- Basic experience with AWS services (S3
- Lambda, or DynamoDB)
- Experience using Git for version control and familiarity with Bitbucket/GitHub
- Foundational knowledge of SQL or NoSQL databases and how to perform basic data operations
- Exposure to unit testing frameworks (Vitest
- Jest, or Cypress)

## Responsibilities
- Design, develop, and maintain modern web applications under the guidance of senior leadership
- Work closely with cross-functional teams to ensure technical feasibility and a seamless user experience
- Troubleshoot and resolve software defects while implementing feature enhancements based on user feedback
- Proactively research and adopt new technologies and frameworks to meet evolving project needs
- Break down complex requirements and provide accurate development estimates
- Write clean, maintainable source code and robust unit tests aligned with technical designs
- Actively participate in code reviews to learn best practices and ensure high-quality output
- Contribute to clear and concise technical documentation
