# Sr. Data Engineer

**Company:** Capstone Logistics, LLC
**Location:** Peachtree Corners, GA
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## Core Skills
- Data Engineering
- ETL
- Python
- PySpark
- SQL

## Minimum qualifications
- Bachelor's degree in computer science
- Engineering, or a related field
- Proficiency in Python (PySpark)
- SQL
- Strong hands-on experience with Microsoft Azure services
- Azure DevOPS
- Experience with Delta Lake
- Spark SQL, and Databricks Jobs
- 4+ Heavy Data experience across the full data life cycle
- Experience in architecting enterprise-wide data environments and platforms
- Heavy experience in data movement, transformation, and modeling processes (ELT/ETL)
- 4+ years' experience working with large datasets or database systems
- 2+ Years working with cloud data technologies

## Preferred qualifications
- AWS/Azure Experience
- MPP technology Experience (Snowflake/Synapse/Redshift)

- Develop and maintain batch/streaming data pipelines using Azure Databricks
