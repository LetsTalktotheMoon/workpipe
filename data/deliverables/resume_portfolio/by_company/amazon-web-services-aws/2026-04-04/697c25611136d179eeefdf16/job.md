# Software Development Engineer II, AI/ML Elastic Collectives - Annapurna Labs

**Company:** Amazon Web Services (AWS)
**Location:** Cupertino, CA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Amazon Web Services (AWS) is seeking an experienced engineer to work on distributed AI/ML systems within Annapurna Labs. The role involves developing collective operations that enable AI to scale across multiple accelerators and servers, focusing on networking solutions for Machine Learning and High-Performance Computing workloads.

## Core Skills
- Software Development
- Software Design
- Software Architecture
- Linux
- Python

## Minimum qualifications
- 3+ years of non-internship professional software development experience
- 2+ years of non-internship design or architecture (design patterns, reliability and scaling) of new and existing systems experience
- Experience programming with at least one software programming language
- Knowledge of Linux fundamentals

## Preferred qualifications
- 3+ years of full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations experience
- Bachelor's degree in computer science or equivalent
- Knowledge of Python and/or C++ programming

## Responsibilities
- Work on distributed AI/ML systems
- Develop collective operations that enable AI to scale across multiple accelerators & servers
- Focus on building networking solutions for Machine Learning (ML) and High-Performance Computing (HPC) workloads on AWS
- Collaborate with infrastructure experts, hardware engineers
- RTL engineers, scientists & architects
- Provide mentorship to new and junior engineers
