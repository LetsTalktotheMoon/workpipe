# Data Scientist

**Company:** Medidata Solutions
**Location:** New York, United States
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Medidata Solutions is a leader in powering smarter treatments and healthier people through digital solutions for clinical trials. The Data Scientist role involves designing AI-driven features, developing machine learning models for clinical trial applications, and building end-to-end machine learning pipelines.

## Core Skills
- Machine Learning
- Classification
- Regression
- Feature Selection
- Python

## Minimum qualifications
- Master's degree in Math
- Statistics
- Data Science
- Computer Science
- Physics
- Bioinformatics or another quantitative field with a foundation in statistics
- Minimum of 2 years of experience
- Proficiency with machine learning techniques (e.g., classification, regression, and feature selection)
- Proficiency using Python
- SQL
- Linux shell scripting
- AWS
- Docker, and Git

## Preferred qualifications
- Experience with LLM or transformer-based algorithms and methods
- Familiarity with GPUs, deep learning, cloud-based deployment, and software development
- Familiarity with machine learning infrastructure and frameworks
- CI/CD, and MLOps
- Experience with clinical trial data or large healthcare datasets

## Responsibilities
- You are a Data Scientist who will design new AI-driven features integrated with Medidata products
- Develop and validate machine learning models for novel clinical trial applications
- Develop prototypes that communicate how models can be used within customer-facing products
- Evaluate and assess novel tools, algorithms, and technologies to be an AI community enabler
- Build end-to-end machine learning pipelines from data curation, processing, model building, model evaluation, to model deployment for production
