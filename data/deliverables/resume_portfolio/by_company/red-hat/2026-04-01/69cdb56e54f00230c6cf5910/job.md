# Senior Software Engineer - Konflux

**Company:** Red Hat
**Location:** Raleigh, NC
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Red Hat is the world’s leading provider of enterprise open source software solutions, using a community-powered approach to deliver high-performing Linux, cloud, container, and Kubernetes technologies. The Senior Software Engineer will lead the design, development, and implementation of software components while collaborating with cross-functional teams to improve system performance and functionality.

## Core Skills
- Golang
- Java
- Python
- Source control management
- Linux development

## Minimum qualifications
- Ability to work with new and fast-changing technologies with a high level of uncertainty, reacting to changes in requirements quickly
- Strong problem-solving skills, with a passion for learning and adapting to new challenges
- Bachelor's degree in Computer science or equivalent working experience
- At least 4 years of experience writing code in Golang
- JAVA, or Python
- Experience working on shared codebases using source control management (SCM) systems like GitHub or GitLab
- Excellent written and verbal communication skills in English and ability to work effectively in a team-oriented environment
- Experience working with Linux as a developer
- Experience with building & debugging applications on Kubernetes clusters
- Experience building and running containerized applications and controllers
- Experience with cloud-based infrastructure (e.g.
- AWS) and microservices architecture

## Preferred qualifications
- Experience with IBM cloud
- Experience in Terraform
- Experience in Tekton
- Experience in CD tools, e.g. ArgoCD
- Experience building CICD pipelines
- Experience in Observability tools such as Prometheus
- Grafana
- Splunk
- Experience in Open Source development
- Experience defining requirements between application components
- Experience in DevSecOps or Product Security
- Experience with OpenShift

## Responsibilities
- Lead the design, development, and implementation of software components, systems, and sub-features
- Collaborate with cross-functional teams on deliverables to improve system performance and functionality
- Exercise good judgment and work with minimal instruction to resolve moderately complex technical issues
