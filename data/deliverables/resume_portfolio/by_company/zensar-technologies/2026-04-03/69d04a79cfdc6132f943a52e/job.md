## About the job
Zensar Technologies is a company focused on building relationships and fostering employee happiness. They are seeking a Data Engineer who will be responsible for developing and implementing data pipelines, analyzing SQL queries, and working with various data warehousing technologies.

- RDBMS concepts
- SQL queries

- DB2

## Minimum qualifications
- Strong background in programming
- RDBMS concepts
- SQL queries
- Procedures and general software development
- Expert at analyzing and developing queries in SQL in various dialects (SQL Server
- DB2
- Oracle)
- Individual contributor and ability to understand existing code without much learning curve of the underlying technology
- Has developed and implemented integrations between applications via API and other web
- Capable of writing ETL pipelines using Python from scratch
- Expert in OOP principles and concepts
- Ability to independently write efficient and reusable code for ETL pipelines
