# Data Scientist

**Company:** Tubi
**Location:** San Francisco, CA (Hybrid)
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Tubi is a free streaming service that entertains over 100 million monthly active users. They are seeking a Data Scientist to build machine learning algorithms and data pipelines, deploy models to production, and conduct thorough testing of models to validate performance and implement improvements.

## Core Skills
- Machine Learning
- Data Pipelines
- Python
- Java
- NumPy

## Minimum qualifications
- Bachelor's degree or foreign equivalent in Data Science
- Physics, or related quantitative field
- Three (3) years of experience in the job offered or related occupation
- Experience with common tools for building and deploying machine learning model and applications: Python
- Java
- NumPy
- Pandas
- Scala/Spark
- XGBoost, sklearn
- TensorFlow
- MLflow
- Writing production-quality code, incorporating testing, evaluation, and monitoring
- ETL with AWS Glue and AWS Cloud Watch monitoring
- Docker deployments

## Responsibilities
- Build machine learning algorithms and data pipelines, deploy models to production, and test models online and offline in A/B test environments
- Conduct thorough testing of models in live environments using A/B testing methodologies to validate performance and iterate on improvements
- Collaborate with data scientists, growth engineers, data engineering, product managers, and other teams to ensure algorithms can be launched successfully
- Deploy machine learning models into production ensuring reliability, scalability, and maintainability
- Create and utilize systems to monitor and analyze the performance of deployed models, identifying opportunities for optimization and enhancement
- Drive new initiatives to help the growth marketing team use machine learning to optimize different applications including email campaigns, push notifications, user acquisition strategies, and personalized recommendations
