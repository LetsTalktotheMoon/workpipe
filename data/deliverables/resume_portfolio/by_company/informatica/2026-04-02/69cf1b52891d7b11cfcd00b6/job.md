# Network Automation Engineer

**Company:** Informatica
**Location:** CA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Informatica is seeking a Network Automation Engineer to help modernize and automate their global enterprise network infrastructure. The role involves designing and building automation solutions to manage network infrastructure through code and collaborating with network engineers to improve operational efficiency and reliability.

## Core Skills
- Network automation
- Python
- Ansible
- Cloud-native networking
- AWS

## Minimum qualifications
- Experience with network automation using Python and Ansible
- Experience working with cloud-native networking environments, particularly AWS
- Strong understanding of network fundamentals, including routing, switching
- IP addressing, and network security concepts
- Experience working with APIs and structured data formats such as JSON
- YAML, or XML
- Experience using version control systems such as Git in collaborative development environments

## Preferred qualifications
- Infrastructure-as-code tools such as Terraform
- CI/CD pipelines or DevOps workflows supporting infrastructure automation
- Enterprise networking platforms such as Cisco environments
- DNS
- DHCP, and IPAM solutions
- Observability or network monitoring platforms
- Multi-cloud networking environments
- A related technical degree required
- Experience working in large-scale enterprise network environments
- Familiarity with DevOps or NetDevOps operating models
- Industry certifications such as Cisco CCNA or higher are a plus

## Responsibilities
- Build and maintain network automation frameworks that enable infrastructure configuration and operations to be managed through code
- Develop automation workflows that support network provisioning, configuration management, and operational tasks
- Collaborate with network engineering teams to translate standard network configurations into reusable automation modules
- Develop integrations using APIs and infrastructure automation tools to streamline network operations
- Contribute to Git-based development workflows and CI/CD pipelines that support infrastructure automation
- Maintain development and testing environments to validate automation changes before production deployment
- Improve operational efficiency and reliability by identifying manual processes and automating them
- Troubleshoot network infrastructure and automation workflows to support a stable and scalable global network environment
