# Software Engineer - Core Services

**Company:** Jewelers Mutual Group
**Location:** Raleigh, NC
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Jewelers Mutual Group is a technology-focused organization committed to supporting the Jewelry industry since 1913. They are seeking a Software Engineer to work on the Core Services squad, which owns the foundational platform for their engineering organization, focusing on expanding the platform with AI-assisted development and resilient service patterns.

## Core Skills
- TypeScript
- Backend services
- Cloud environment
- Event-driven architecture
- Messaging systems

## Minimum qualifications
- 2–4 years of software engineering experience, with meaningful exposure to backend services in a cloud environment
- Comfort with TypeScript or a strong willingness to go deep on it quickly: this is our primary language
- Familiarity with event-driven architecture and messaging systems
- Kafka or similar experience is a plus
- Infrastructure fluency: you understand what your code runs on and can write or modify Terraform when needed
- Security-conscious by default: you think about auth, least-privilege access, and data sensitivity without being prompted
- AI-forward mindset: you use AI tooling in your development workflow and think actively about where agentic approaches can replace manual work
- Strong written and verbal communication: this squad works across team boundaries constantly

## Responsibilities
- TypeScript Lambda microservices and API Gateway integrations across our AWS-native stack
- Event-driven workflows using Apache Kafka (via Redpanda Managed) for asynchronous, decoupled system communication
- Infrastructure as code with Terraform and Spacelift, owning deployments from first commit to production
- Resilient service patterns: retry logic, dead-letter queues, graceful degradation, idempotent handlers: that make the platform boring in the best way
- Agentic AI solutions using Claude Code and the Anthropic API, including automated workflows and intelligent processing pipelines
- Shared libraries, tooling, and internal standards that raise the quality floor for the entire engineering organization
