# Software Engineer

**Company:** Motorola Solutions
**Location:** Allen, TX
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Motorola Solutions is a global community focused on keeping people safer through technology. The Software Engineer will understand complex network and application infrastructure, troubleshoot issues, and build tools to improve application performance and reliability in cloud environments.

## Core Skills
- Network Infrastructure
- Application Infrastructure
- Monitoring Tools
- Troubleshooting
- Operating Systems

## Minimum qualifications
- Masters degree or Bachelors degree + 2 years experience

## Responsibilities
- Understand complex network and application infrastructure and build tools that helps engineers to monitor, triage and troubleshoot issues from the network all the way through the application stack in public and private cloud environments
- Troubleshoot complicated, cross platform issues handling OS
- Networking
- Database in a cloud-based SaaS environment and handle live production incidents, debug/troubleshoot application and infrastructure issues, follow and implement SRE best practices
- Gain and disseminate deep knowledge of push to talk communications solutions provided by Motorola solutions for ensuring reliability and maintainability of hardware, software processes, network facilities, controls, and security systems leveraged by the product
- Monitor application performance, take steps to improve overall application performance and stability and follow through with implementation
- Own end-to-end availability and performance of key network and application services and build automation to prevent problem recurrence
- Automate response to all non-exceptional service conditions
- Perform end to end testing to ensure the overall quality of Mission Critical Push to talk System
- Conduct system analysis, configuration management and develops improvements for system software performance, availability and reliability
