# Software Engineer, Backend

**Company:** Aura
**Location:** United States
**Seniority:** Mid Level
**Work Model:** Remote
**Employment Type:** Full-time

## About the job
Aura is on a mission to create a safer internet, providing intelligent digital safety products to protect customers against digital threats. The Software Engineer (Backend) will focus on building and maintaining high-performance backend systems, integrating AI-driven components, and managing infrastructure with infrastructure-as-code.

## Core Skills
- Python
- Backend Development
- Linux
- Containers
- Docker

## Minimum qualifications
- 4+ years of experience in software engineering or product engineering
- Strong proficiency in Python for backend development
- Experience working with Linux, containers (Docker
- Kubernetes, etc.), and cloud environments
- Knowledge of AWS cloud services
- Git-based workflows for development, including pull requests and code reviews
- Bachelor's or Master's degree in Computer Science
- Engineering, or a related quantitative field, or equivalent industry experience

## Preferred qualifications
- Experience with Databricks for large-scale data processing
- Hands-on experience with Terraform for infrastructure management
- Familiarity with GitHub Actions for CI/CD automation
- Background in AI/ML technologies and deploying ML models in production

## Responsibilities
- Develop and enhance backend features, ensuring system reliability and scalability
- Collaborate with stakeholders to define requirements and improve system performance
- Manage infrastructure using Terraform and other infrastructure-as-code tools
- Work closely with Git-based workflows, including code reviews and contributions
- Key Projects: Integrating with LLM Agents and building features around them
- Hosting and optimizing ML models to serve millions of requests per day
- Processing and managing large-scale data efficiently, ensuring low-latency client access
