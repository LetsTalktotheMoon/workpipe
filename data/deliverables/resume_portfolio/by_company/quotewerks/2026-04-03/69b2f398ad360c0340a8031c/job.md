# Software Developer

**Company:** QuoteWerks
**Location:** Orlando, FL
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
QuoteWerks is a Florida-based software company dedicated to helping small and mid-sized businesses streamline operations and drive growth through quote and proposal software. They are looking for a Software Developer who will build and maintain business-critical web applications used daily by customers. This role involves designing, developing, testing, and supporting web applications while collaborating with engineers and stakeholders.

## Core Skills
- C#, .NET development
- VB.NET
- JavaScript
- Web APIs

## Minimum qualifications
- Strong experience with .NET development (C#
- VB.NET) and JavaScript, including building and maintaining full-stack web applications using technologies like Web APIs
- AngularJS
- HTML/CSS, and SQL databases
- Hands-on experience designing, integrating, and working with APIs, including 3rd-party integrations, and maintaining/evolving legacy systems
- Solid understanding of software architecture, design principles, and source control tools (e.g.
- Team Foundation Server)
- Curiosity and willingness to learn new technologies

## Preferred qualifications
- Python experience is a plus
- Exposure to modern frontend frameworks, cloud/DevOps practices, or AI-assisted development tools

## Responsibilities
- Design, develop, test, deploy, and support business-critical web applications, working across the full stack including databases, backend services/APIs, and web-based user interfaces
- Contribute to the migration of functionality from QuoteWerks Desktop to QuoteWerks Web, refactoring and re-architecting features as needed to support scalability, performance, and maintainability
- Work within a mature legacy codebase, addressing technical debt while also contributing to new systems and next-generation platform initiatives using modern development practices
- Collaborate with engineers and stakeholders to estimate, prioritize, and deliver work, managing individual project timelines and leveraging modern tooling (including AI-assisted development) to improve code quality and engineering efficiency
