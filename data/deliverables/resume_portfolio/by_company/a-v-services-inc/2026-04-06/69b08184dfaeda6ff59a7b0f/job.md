# Software Engineer II

**Company:** A-V Services, Inc.
**Location:** New York, NY
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
A-V Services Inc. is a leading AV company with multiple fortune 100 contracts within the financial, pharmaceutical, and computing industry, seeking a Software Engineer II. The role involves designing and delivering technology products, implementing solutions across diverse technical domains, and supporting various business functions to achieve strategic goals.

## Core Skills
- Software Engineering
- System Design
- Application Development
- Testing
- Operational Stability

## Minimum qualifications
- Formal training or certification on software engineering concepts and 3+ years' applied experience
- Hands-on practical experience in system design, application development, testing, and ensuring operational stability
- Experience with front-end technology React
- Proficiency in developing RESTful APIs using Python Fast API
- Experience with version control systems, particularly GIT
- Strong knowledge of SQL and experience in database design and development
- Solid understanding of agile methodologies such as CI/CD
- Application Resiliency, and Security
- Demonstrated knowledge of software applications and technical processes within a technical discipline (e.g., cloud, artificial intelligence, machine learning, mobile, etc.)
- Experience with application monitoring and creating dashboards using Splunk
- Grafana, and Datadog

## Preferred qualifications
- Familiarity with cloud platforms, such as Cloud Foundry
- Strong communication skills and a willingness to learn

## Responsibilities
- Develop and maintain backend services using Python
- Collaborate with the front-end team to integrate ReactJS applications
- Work with SQL databases to design, query, and manage data
- Assist in the development and maintenance of applications using .NET
- Participate in code reviews and contribute to team knowledge sharing
- Stay updated with the latest industry trends and technologies
