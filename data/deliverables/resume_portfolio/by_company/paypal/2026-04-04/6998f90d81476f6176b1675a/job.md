# Software Engineer - Java

**Company:** PayPal
**Location:** San Jose, CA
**Seniority:** Entry Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
PayPal has been revolutionizing commerce globally for more than 25 years. As a Backend Software Engineer (Java), you’ll collaborate with experienced engineers to design, develop, and optimize backend components that ensure performance, reliability, and scalability across our global payment platform.

## Core Skills
- Java
- Spring Boot
- Micronaut
- RESTful web services
- Service-Oriented Architecture

## Minimum qualifications
- 1+ years relevant experience and a Bachelor's degree OR Any equivalent combination of education and experience
- Strong foundation in programming concepts, algorithms, and data structures
- Experience with backend development using Java and frameworks such as Spring Boot or Micronaut
- Understanding of RESTful web services and Service-Oriented Architecture (SOA) principles
- Familiarity with relational and NoSQL databases (e.g.
- MySQL
- PostgreSQL
- MongoDB
- Cassandra)
- Exposure to version control systems (e.g.
- Git) and collaborative development workflows (e.g.
- Agile
- CI/CD)

## Preferred qualifications
- Interest in large-scale, high-performance distributed systems
- Awareness of the payment processing industry and related compliance or regulatory standards
- Exposure to cloud platforms such as AWS
- GCP, or Azure
- Contributions to open-source projects or participation in developer communities

## Responsibilities
- Implements tasks within the Software Development Lifecycle (SDLC), receiving structure and oversight from more experienced staff
- Follows well-established internal conventions and standard procedures
- Understands internal standards & processes and applies them to make technical decisions
- Collaborates with peers, manager, and project lead to gain understanding of tasks and review solutions
- May contribute to code & design reviews
