# Software Development Engineer, Amazon Connect Contact Lens

**Company:** Amazon Web Services (AWS)
**Location:** Seattle, WA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Amazon Web Services (AWS) is a leader in cloud computing and is seeking Software Development Engineers for their Applied AI Solutions organization. The role involves developing innovative technology solutions for Amazon Connect, an AI-powered customer experience solution, focusing on building modern contact centers and delivering scalable systems.

## Core Skills
- Software Development
- Software Design
- Software Architecture
- Programming Languages
- Software Development Life Cycle

## Minimum qualifications
- 3+ years of non-internship professional software development experience
- 2+ years of non-internship design or architecture (design patterns, reliability and scaling) of new and existing systems experience
- Experience programming with at least one software programming language

## Preferred qualifications
- 3+ years of full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations experience
- Bachelor's degree in computer science or equivalent

## Responsibilities
- Own code end-to-end from development
- Work closely with senior technical leaders within the team and across the organization
- Develop the next generation contact center
- Deliver scalable, resilient systems while maintaining a constant customer focus
