# Software Engineer - Machine Learning (FC QVE)

**Company:** Electronic Arts (EA)
**Location:** Orlando, FL
**Seniority:** Entry Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Electronic Arts (EA) creates next-level entertainment experiences that inspire players and fans around the world. They are seeking an enthusiastic Software Engineer with a focus on Machine Learning to build predictive models and targeted ML solutions for their FC franchise. The role involves implementing ML designs, predicting code change impacts, and collaborating with Quality Engineers and central AI/ML teams.

## Core Skills
- Machine Learning
- Python
- C++
- Backend development
- System design

## Minimum qualifications
- BSc in Computer Science
- Engineering, or equivalent experience
- 1-2 years of software engineering or ML experience
- Proficiency in Python
- C++, or other backend languages
- Knowledge in ML engineering, including model training, evaluation, and continuous learning
- Understanding of backend development, system design, and API development
- Collaborative skills with the ability to partner across multiple teams, supported by strong written and verbal communication
- Experience using AI-assisted coding tools to accelerate development

## Responsibilities
- Help build machine learning models that learn from multiple data to predict code change impacts and help engineers make safer, high-quality changes faster
- Train models using diverse data sources to forecast and ensure realistic in-game simulations and identifies potential issues before release
- Help design LLM-driven automated testing frameworks to verify state changes at scale
- Partner with embedded Quality Engineers and central AI/ML teams to integrate targeted ML models into local developer workflows
