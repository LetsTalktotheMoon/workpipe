# Software Engineer - Automation Tools and Infrastructure

**Company:** Zoox
**Location:** Foster City, CA
**Seniority:** Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Zoox is developing the first ground-up, fully autonomous vehicle fleet and is seeking a Software Engineer to join their Firmware Tools team. In this role, you will write software to integrate tools for automated firmware prototyping, development, testing, and integration.

## Core Skills
- HIL Test Frameworks
- SIL Test Frameworks
- Object-oriented programming
- Python
- C++

## Minimum qualifications
- BS in relevant engineering discipline and 1+ years of relevant work experience
- HIL and/or SIL Test Frameworks
- Object-oriented programming experience (Python or C++ preferred)
- Experience building cross-platform software with source control (Git)
- Strong interpersonal and communication skills

## Preferred qualifications
- CI and build systems (e.g. Bamboo
- Buildkite, etc.)
- Python libraries (e.g. pybind, pytest, pandas, python-can, plotly, matplotlib, etc.)
- Communication protocols (e.g. CAN
- LIN, pub/sub)
- ClickHouse
- OpenSearch
- Grafana
- Databricks, etc
- Databases and Observability Platforms

## Responsibilities
- Develop software for an automated validation framework, supporting continuous integration (CI) via smoke, sanity, nightly, weekly pipelines
- Research, design, develop, rollout, and maintain software tools and test framework for development and validation
- Integrate tester software systems with server infrastructure, including networking (CAN, wifi, cellular
- WIFI
- NFC, etc), hardware instruments (DAQs
- PSUs), protocol-level simulator/spoofer hardware, etc
- Propose tool and/or feature ideas to the team to improve processes
