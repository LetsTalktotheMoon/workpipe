# Senior Software Engineer- Networking Technologies

**Company:** Cisco
**Location:** Milpitas, California, US
**Seniority:** Mid, Senior Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Cisco is a leading technology company that is revolutionizing how data and infrastructure connect and protect organizations. They are seeking a Senior Software Engineer to develop scalable systems and robust platforms, focusing on the Cisco 8000 series of routers and Silicon One ASIC features.

## Core Skills
- C
- C++
- Python
- Pytest
- Linux Operating Systems

## Minimum qualifications
- Bachelors + 6 years of related experience OR Masters + 4 years of related experience OR PhD + 1 year of related experience
- Programming experience in C/C++
- Experience using Python to build automated test suites for kernel module validation, including experience with frameworks like Pytest
- Experience with Linux Operating Systems and debugging tools
- Experience with core networking technologies, including TCP/IP, routers, switches, and Network Processing Units (NPUs)

## Preferred qualifications
- Understanding of Linux operating systems, development/debugging tools, and memory management
- Experience designing and implementing multi-threaded applications and architectures
- Ability to troubleshoot and resolve complex, system-level issues across multiple integrated components
- Experience with industry-standard debugging tools (e.g.
- GDB) and memory analysis tools (e.g.
- Valgrind)
- Experience leveraging AI-powered tools to enhance coding, code reviews, and testing workflows
- Experience developing and maintaining unit test (UT) automation frameworks
- Knowledge of MACsec
- PCIe, or DMA is considered a plus
- Understanding of tools such as GIT
- Bitbake
- Jira
- Jenkins and CI/CD pipeline tools

## Responsibilities
- Develop and improve software modules within IOS-XR to support NPU or fabric sub-systems
- Contribute to XR platform software that interfaces with the ASICs for platform enablement, interrupt management, health monitoring of the fabric subsystem including performance qualification and resolving issues on the fabric datapath
- Contribute to platform software for security protocols such as MACSec
- Participate in debugging and resolving system-level issues across software and hardware boundaries
- Contribute to the design of scalable and maintainable software components for NPU management
- Participate in unit testing, system testing, and automation efforts to ensure high-quality work
- Conduct code reviews and mentor junior engineers
