# Software Engineer – Backend Integration & Modernization

**Company:** HealthPartners
**Location:** Bloomington, MN, United States
**Seniority:** Entry, Mid, Senior Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
HealthPartners is currently hiring a Software Engineer – Backend Integration & Modernization. This role primarily supports the healthcare claims domain, focusing on modern service development and system integration, connecting IRIS-based core platforms with modern APIs, services, and user experiences.

## Core Skills
- Backend service development
- API design, development
- Java
- Spring Boot

## Minimum qualifications
- Bachelor's degree in Computer Science
- MIS
- Business Administration, or equivalent experience
- Three (3) years of experience developing backend services and APIs
- Strong experience with Java and Spring Boot
- Experience integrating with legacy systems and complex data layers
- Experience designing and consuming REST and/or SOAP services
- Experience using AI-assisted development tools (e.g.
- Claude Code
- GitHub Copilot, or similar) to improve developer productivity, code quality, or documentation
- Familiarity with relational and non-relational databases
- Ability to work across multiple languages and platforms
- Ability and willingness to learn IRIS/ObjectScript concepts and proprietary tools
- Participation in a rotating on call schedule

## Preferred qualifications
- Experience integrating with InterSystems IRIS or similar nontraditional databases
- Experience designing or using agentic AI workflows (e.g.
- AI agents for code generation, refactoring assistance, test creation, or knowledge extraction from legacy systems)
- Familiarity with integrating AI tools into enterprise development workflows, including prompt engineering, guardrails, and human‑in‑the‑loop validation
- Experience working with and/or modernizing legacy systems
- Experience with containerized or cloud-based platforms
- Familiarity with healthcare or insurance systems
- Experience using AI assisted development tools for productivity and code quality

## Responsibilities
- Design and implement backend services that integrate modern applications with IRIS based systems
- Modernize existing interfaces while preserving business correctness and performance
- Define and follow standard API and service design patterns
- Partner with IRIS/ObjectScript engineers to ensure proper data access and transaction handling
- Perform unit and integration testing
- support system and end-user testing
- Troubleshoot and resolve production issues related to integrations and services
- Contribute to documentation and shared development standards
