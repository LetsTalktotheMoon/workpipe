# Technical Data Analyst III

**Company:** GHX
**Location:** United States
**Seniority:** Mid, Senior Level
**Work Model:** Remote
**Employment Type:** Full-time

## About the job
Global Healthcare Exchange (GHX) is a healthcare business and data automation company that empowers healthcare organizations to enhance patient care and maximize savings through advanced technology solutions. They are seeking a Technical Data Analyst III to focus on data analysis, reporting, and data quality, ensuring the integrity of content data through complex SQL development and analytics reporting.

## Core Skills
- SQL
- SQL query optimization
- Relational databases
- Data analysis
- Data profiling

## Minimum qualifications
- Bachelor's degree in Computer Science
- Data Science
- Information Systems, or related degree
- 5+ years of experience writing complex SQL queries including nested joins, temporary tables, aggregations, subqueries, window functions, and CTEs
- 5+ years of experience with relational databases (PostgreSQL
- SQL Server
- MySQL) and SQL query performance optimization
- Experience with data profiling, data quality assessment, data validation techniques, and finding insights in large, complex data sets
- Strong analytical skills including data mining and evaluation, with demonstrated ability to perform code/query reviews in Agile/Scrum environments
- Proficient with multiple technologies including SQL databases, data analysis tools
- BI platforms, and scripting languages
- Expert-level SQL skills with deep understanding of complex queries, query optimization, and performance tuning
- Strong analytical skills including data mining, evaluation, analysis, and visualization
- Identifies, implements, and applies best practices for data analysis and quality
- is the 'go to' person on the team
- Ability to handle multiple projects and possesses a proven track record of high-quality deliverables

## Preferred qualifications
- Hands-on experience with Snowflake data warehouse and Sigma Computing for building analytics reports and dashboards
- Familiarity with healthcare data rules and standards (HIPAA
- HL7) or healthcare supply chain/IT systems

## Responsibilities
- Write and optimize complex SQL queries including nested joins, temp tables, aggregations, window functions, and CTEs
- Identify patterns and relationships to evaluate, maintain, and enrich data quality across multiple systems and data sources
- Perform root cause analysis (RCA) and impact analysis to identify data quality issues and report findings to stakeholders
- Develop flexible, reusable, and efficient SQL code with minimal supervision while adhering to best practices
- Create and maintain analytics reports and dashboards for business stakeholders using modern BI tools
- Conduct data profiling and data quality validation to ensure high-quality data for downstream consumers
- Perform code reviews on SQL and data scripts, providing constructive feedback to ensure quality
- Collaborate with cross-functional teams including data engineering, product, and analytics teams in an Agile/Scrum environment
