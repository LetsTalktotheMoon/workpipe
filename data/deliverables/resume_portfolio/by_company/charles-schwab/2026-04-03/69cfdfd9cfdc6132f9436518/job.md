# Java Developer - Workplace Services Engineering

**Company:** Charles Schwab
**Location:** Southlake, TX
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Charles Schwab is a financial services company focused on innovative solutions for workplace financial well-being. They are seeking a skilled Java Developer to contribute to the design and implementation of multi-platform application systems while collaborating with diverse teams to enhance software quality and delivery.

## Core Skills
- Java
- SQL
- NoSQL
- Spring Framework
- Spring Boot

## Minimum qualifications
- 4+ years of related experience required
- BS/MS degree preferred or equivalent experience
- 3+ years of professional hands-on experience developing and reviewing Java code, with a focus on clean, maintainable design
- Experience working with SQL and NoSQL databases (e.g.
- Oracle
- MongoDB)
- Experience using Spring or Spring Boot frameworks
- Experience designing and implementing RESTful APIs and microservices, ideally in a cloud‑based environment
- Experience developing CI/CD pipelines with tools such as Bamboo
- GitHub Actions
- Jenkins
- CircleCI, or Travis CI
- Understanding of distributed systems, integration patterns (e.g.
- RabbitMQ
- Kafka), and unit testing frameworks (e.g.
- JUnit)
- Strong analytical, problem‑solving, and communication skills with the ability to collaborate effectively in an Agile environment

## Preferred qualifications
- Experience using AI tools to enhance code quality, automate testing, or accelerate development
- Familiarity with cloud‑native development or containerization (Docker
- Kubernetes)
- Experience with monitoring and performance engineering tools (e.g.
- AppDynamics
- Splunk
- Grafana
- DataDog
- PCF metrics)
- Experience with test‑driven development (TDD), behavior‑driven development (BDD), or frameworks such as Cucumber or Selenium
- Front‑end development experience with Angular or React
- Cross‑functional communication and collaboration

## Responsibilities
- Contribute to the design, development, and implementation of cross‑functional, multi‑platform application systems
- Deliver high‑quality, testable code using secure development practices and modern engineering standards
- Participate in constructive and inclusive code reviews that encourage learning and shared ownership
- Build Proof‑of‑Concept solutions and reusable patterns to accelerate development and innovation
- Prepare and maintain architecture, design, and system documentation
- collaborate with architects to ensure alignment and approvals
- Coordinate with onshore and offshore engineering partners in a respectful and collaborative manner
- Develop CI/CD pipelines using tools such as Bamboo and GitHub Actions
- Write and maintain automated unit tests to support high‑quality delivery
- Participate fully in Agile ceremonies and contribute to a culture of continuous improvement and psychological safety
