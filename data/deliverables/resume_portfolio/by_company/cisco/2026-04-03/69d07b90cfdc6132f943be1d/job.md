# Software Engineer

**Company:** Cisco
**Location:** Durham, NC
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Cisco is revolutionizing how data and infrastructure connect and protect organizations in the AI era. They are seeking a Software Engineer to join their Infrastructure and Cloud Services team, focusing on building and improving tools and integrations that support Cisco’s cloud services.

## Core Skills
- Java
- Python
- Go
- JavaScript
- ReactJS

## Minimum qualifications
- 3+ years of hands-on software development in enterprise environments
- Strong technical background in Java
- Python
- Go, and JavaScript (including ReactJS and Redux)
- Proven experience building and deploying microservices-based applications within containerized environments
- Solid understanding of software design patterns
- TDD
- CI/CD pipelines, and RESTful APIs
- Experience with database management (SQL and NoSQL)

## Preferred qualifications
- Bachelor's or Master's degree in Computer Science or a relevant technical field
- Experience deploying and managing applications on platforms such as AWS
- GCP
- VMWare
- OpenStack, or OpenShift
- Proficiency with infrastructure-as-code and automation tools, including Terraform
- Ansible
- GitOps
- Jenkins, and Argo CD
- Experience with monitoring and logging tools (Prometheus
- Grafana
- ELK
- Fluent Bit) and a strong grasp of secure software development practices
- Demonstrated ability to mentor junior engineers, conduct thorough code reviews, lead postmortem analysis, and share technical knowledge through documentation, blogs, or internal presentations

## Responsibilities
- Design and develop scalable, microservices-based applications using Java
- Python
- Go, and JavaScript, ensuring high-quality code through rigorous TDD practices and modern software design patterns
- Architect and implement robust CI/CD pipelines and infrastructure-as-code (IaC) solutions using Terraform
- Ansible, and GitOps to streamline continuous delivery and deployment
- Deploy, manage, and optimize enterprise applications across multi-cloud and hybrid environments, including AWS
- GCP
- VMWare, and OpenShift, to ensure high availability and performance
- Monitor production environments using tools like Prometheus
- Grafana, and the ELK stack, serving as an escalation point for on-call issues and driving proactive incident resolution
- Lead postmortem reviews and conduct comprehensive root cause analysis (RCA) to identify system bottlenecks, improve reliability, and prevent future production incidents
- Develop and maintain secure, high-performance RESTful and GraphQL APIs, incorporating OAuth 2.0 and OpenAPI 3.0 standards to ensure robust integration across systems
- Mentor junior engineers, conduct thorough code reviews, and provide technical guidance to foster a culture of engineering excellence and continuous learning within the team
- Collaborate with cross-functional partners to estimate project timelines, track milestones, and provide ongoing updates, while sharing technical knowledge through documentation, internal presentations, and blogs
