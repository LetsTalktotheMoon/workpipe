# Cloud Operations Senior

**Company:** CLA (CliftonLarsonAllen)
**Location:** Raleigh, NC
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
CLA is a top 10 national professional services firm dedicated to creating opportunities for clients, people, and communities. They are seeking an experienced Cloud Operations Senior to manage, monitor, and maintain cloud environments, ensuring optimal performance and security.

## Core Skills
- Kubernetes
- Azure
- Cloud Security
- Linux
- Python scripting

## Minimum qualifications
- 4 years of relevant experience required
- Bachelor's degree or a combination of relevant experience and training may be considered in lieu of a degree
- Proficiency in Manage Kubernetes
- Azure
- Cloud Security
- Linux using Python or Bash scripting
- PowerShell
- Docker
- Git
- DevOps
- Spacelift
- M365 suite
- EntraID
- Hyperconverged systems
- Windows and Linux OS
- Support of both Datacenter and Cloud environments

## Responsibilities
- Monitor and maintain cloud systems to ensure optimal performance and availability
- Automate operational tasks and processes using Infrastructure as Code (IaC) principles
- Monitor and manage cloud resources to ensure scalability and availability
- Collaborate with development teams to deploy applications and services in the cloud
- Perform regular system audits and generate reports on cloud systems health
- Troubleshoot and resolve cloud systems issues
- Document operational procedures and best practices
- Implement best practices for cloud security and compliance
- Manage and monitor on-premises data center operations to ensure seamless integration with cloud systems
- Perform maintenance and upgrades on on-premises data center hardware and software
