# FullStack Software Engineer

**Company:** ModMed
**Location:** Boca Raton, FL
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
ModMed is a healthcare technology company focused on modernizing the healthcare experience through innovative software solutions. They are seeking a FullStack Software Engineer to join their Product Development team, responsible for designing and developing healthcare software systems using technologies like React.js and PHP.

## Core Skills
- React.js
- Redux
- React Router
- Zustand
- Recoil

## Minimum qualifications
- Minimum 3 years of professional web development experience
- Bachelor's Degree in Computer Science or related field
- Strong proficiency in React.js and related libraries (Redux
- React Router
- Zustand
- Recoil)
- Expertise in PHP and frameworks like Laravel or Symfony
- experience with Perl or Python a plus
- Solid understanding of web standards
- APIs
- REST
- XML, and JSON
- Strong SQL skills and experience with relational databases (MySQL preferred)
- Effective communicator with strong collaboration skills
- Strong problem-solving abilities
- adaptable, proactive, and ownership-driven
- Commitment to code quality, architecture, and documentation

## Preferred qualifications
- Experience with Agile methodologies
- Familiarity with design patterns

## Responsibilities
- Work within an agile, cross-functional team to deliver organizational objectives
- Develop user-facing features using React.js and related libraries (e.g.
- Redux
- React Router
- Zustand
- Recoil)
- Make sound design decisions and tradeoffs within the codebase
- Contribute across the codebase with minimal guidance
- Build reusable, efficient, and scalable components and front-end libraries
- Translate UI/UX wireframes into responsive, high-quality code
- Use user empathy to inform design and development decisions
- Collaborate with QA and advise on regression risks
- Proactively fix issues and address technical debt
- Ensure cross-browser compatibility and consistent visual design
