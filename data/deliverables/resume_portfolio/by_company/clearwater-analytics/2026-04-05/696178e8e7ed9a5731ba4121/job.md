# Software Development Engineer III

**Company:** Clearwater Analytics
**Location:** New York, NY
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Clearwater Analytics is a financial technology company seeking a Software Development Engineer III to join their Front Office Suite engineering team. The role involves designing, developing, and testing essential components for the Beacon platform while working closely with clients in the financial sector.

## Core Skills
- Software engineering
- Financial instruments
- Python
- Data structures
- Algorithms

## Minimum qualifications
- 3-5 years of software engineering experience
- Work experience with financial instruments
- High proficiency in Python
- Fundamentals of software engineering such as data structures and algorithms
- Exposure to database programming (SQL)
- Strong problem-solving skills

## Preferred qualifications
- Familiarity with streaming technologies like Kafka and Redis
- Familiarity with SQL and NoSQL database technologies
- Familiarity with cloud computing such as AWS or Azure
- Exposure to multiple programming languages and paradigms

## Responsibilities
- Work on developing, testing, and supporting various components to support our clients in the financial technology space
- These include: Market data ingestion
- Time series framework
- Trade ingestion/high frequency trades
- ETL/Data pipelines
- Integrations with third-party systems
- Adding features to Beacon’s framework for easy developing and deploying of applications
- Providing ongoing support and enhancement of the Beacon Platform
