# Software Development Engineer

**Company:** Adobe
**Location:** San Jose, CA
**Seniority:** New Grad, Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Adobe is looking for a talented and highly motivated Software Engineer to help build and deliver their next-generation Adobe Acrobat Sign service. The role involves working on both front-end and back-end solutions, collaborating with various teams to design and develop a user-friendly web application.

## Core Skills
- HTML
- TypeScript
- JavaScript
- ReactJS
- Web Components

## Minimum qualifications
- B.S. or M.S. in Computer Science or equivalent
- 0-3 years of professional experience developing interactive web applications
- Ability to work independently and strong communication skills within and across teams
- Ability and willingness to learn new technologies quickly with a strong interest in full track development
- High proficiency in developing software solutions using web application client-tier technologies, including HTML and typescript/JavaScript
- ReactJS,/Web components and React Native
- Strong knowledge of server-side Java/J2EE technologies, including Spring
- NodeJS
- Hibernate
- JSP
- JNDI
- JDBC
- JMS
- Servlets, and XML
- Ability to perform independently in a hybrid work environment supported by competent written and verbal communication skills
- Experience with microservices infrastructure (e.g. Docker
- Container Orchestration [ideally Kubernetes])
- Amazon Web Services (AWS) and/or Microsoft Azure

## Responsibilities
- Work hand-in-hand with, engineers, user experience designers and product managers to prototype and build a next-generation web application
- Provide technical leadership to solve ambitious design problems to deliver best-in-class user experience
- Design, deploy, and support secure, efficient, containerized software for Adobe Acrobat Sign
- Design & develop key user-facing features, collaborating with the product and the design teams to work on the most optimized and delightful user experience
- Work with other Adobe teams to explore diverse technologies and embrace cross-team standards
