# Fullstack/ Backend Engineer

**Company:** Qualitest
**Location:** Mountain View, CA
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Qualitest is a leading AI-powered Quality Engineering Company seeking a Full-Stack/Backend Engineer to join their team in Menlo Park, CA. The role involves designing, building, and operating backend services and product surfaces that power internal tools and AI-driven workflows.

## Core Skills
- Python
- TypeScript
- React
- Backend systems development
- API development

## Minimum qualifications
- Strong hands-on production experience building backend systems in Python
- Solid proficiency in TypeScript and modern front-end development using React
- Experience building and maintaining internal tools
- APIs, or product workflows end-to-end
- Practical experience with cloud-native engineering, including: Kubernetes
- Terraform
- AWS CDK
- Containers/containerization
- CI/CD
- Hands-on experience building or integrating GenAI systems in real product or internal tooling contexts
- Familiarity with one or more of the following: Model serving/inference workflows
- Evaluation/monitoring pipelines
- Orchestration/workflow systems
- RAG pipelines
- Agent-style system integrations
- Strong software engineering fundamentals, including: Code quality
- Testing
- Debugging
- Performance awareness
- Maintainable system design

## Preferred qualifications
- Experience building mobile applications or product surfaces for: Android, iOS
- React Native
- Experience delivering GenAI systems to production or near-production environments
- Strong familiarity with orchestration and workflow systems for AI-enabled applications
- Experience operating Kubernetes and Terraform in production environments
- Experience integrating internal services with: Observability systems
- Security systems
- Storage systems
- Data pipelines
- Proficiency in Rust for high-performance or systems-oriented backend development
- Strong product instincts and the ability to move fluidly between prototyping and production-minded implementation

## Responsibilities
- Design, implement, and maintain backend APIs and internal services using Python
- Contribute to service architecture, reliability, and maintainability for internal product workflows
- Build integrations with internal and external systems, including data, storage, authentication, and service layers
- Where appropriate, contribute to performance-sensitive backend components, with Rust as a plus
- Build and maintain user-facing interfaces and internal tools using TypeScript
- React, and related modern web technologies
- Design and implement polished internal product experiences with strong usability and practical workflow support
- Partner on product implementation across desktop and web surfaces
- Support or build high-priority mobile product surfaces across Android, iOS, and/or React Native
- Help extend internal tooling and workflows into mobile-first experiences where needed
- Collaborate on practical mobile UI architecture and implementation decisions
