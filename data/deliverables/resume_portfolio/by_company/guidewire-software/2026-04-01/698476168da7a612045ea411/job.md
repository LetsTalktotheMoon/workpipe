# Software Engineer III - AI Services

**Company:** Guidewire Software
**Location:** San Mateo, CA
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Guidewire Software is a leading platform for P&C insurers, and they are seeking a Software Engineer III to join their team in building a next-generation AI platform. The role involves developing and maintaining features for the application platform, focusing on backend services while ensuring quality, scalability, and security.

## Core Skills
- Backend Software Engineering
- Platform
- Web Services Development
- Coding Agents
- Automation Tools

## Minimum qualifications
- 4-6 years of experience in backend software engineering, with experience building platform or web services
- Experience with coding agents and automation tools
- Proficiency in one of Java
- JavaScript
- TypeScript
- Python, with willingness to learn other languages
- Experience with Large Language Models (LLMs)
- Retrieval-Augmented Generation (RAG), vector databases (vector DBs)
- AI agents, and Model Context Protocol (MCP)
- Good collaboration and communication skills
- Exposure to Kubernetes and infrastructure as code
- Demonstrated ability to embrace AI and apply it to your role as well as use data-driven insights to drive innovation, productivity, and continuous improvement

## Responsibilities
- Develop and maintain features for Guidewire's application platform, primarily focusing on backend services but with flexibility to occasionally work with infrastructure and frontend
- Apply agentic coding tools as Guidewire uses them to dramatically accelerate development
- Write clean, well-tested code with attention to quality, scalability, availability, and security
- Collaborate with team members to deliver secure, scalable, and efficient platform operations, supporting Guidewire's mission of operational excellence and transformative innovation
- Participate in code reviews and contribute to team best practices
