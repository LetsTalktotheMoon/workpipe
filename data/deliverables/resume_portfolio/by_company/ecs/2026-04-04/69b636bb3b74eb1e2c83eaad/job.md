# Full Stack Software Developer

**Company:** ECS
**Location:** Greater Richmond Region
**Seniority:** Mid, Senior Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
ECS is a leading mid-sized provider of technology services to the United States Federal Government. They are seeking a Full Stack Software Developer to work closely with clients to understand needs and implement technical solutions across various programming languages and cloud platforms.

## Core Skills
- Cloud Environments
- Software Development
- Git
- Agile/Scrum
- Customer Experience (CX)

## Minimum qualifications
- Bachelor's degree in engineering or technical discipline, or associates degree with equivalent years of experience
- At least 5 years of overall experience and knowledge of Cloud Environments
- At least 5 years of overall experience and knowledge of Software Development

## Preferred qualifications
- Strong understanding of Git
- Familiarity working in an Agile/Scrum development environment
- Familiarity with CX and UI/UX practices
- Experience with Golang
- C#
- Java
- Python
- C++ or other object-oriented programming languages
- Experience with JavaScript
- CSS
- HTML
- Angular
- Vue
- React
- TypeScript or other front end languages and frameworks
- Experience with AWS and Azure Cloud Environments
- Experience with containerization and microservice architectures

## Responsibilities
- Create innovative solutions to problems, promoting reusability and scalable processes
- Leverage expertise to advise and build solutions that are simple, secure, and maintainable
- Build, maintain, and update automation code and/or scripts
- Develop Rich front-end experiences in concert with back-end services and platforms
- Find opportunities to use data and metrics to inform decisions and create changes to improve the environment
- Design, build and maintain scalable back-end services and APIs, ensuring reliable data flow, security, and performance
