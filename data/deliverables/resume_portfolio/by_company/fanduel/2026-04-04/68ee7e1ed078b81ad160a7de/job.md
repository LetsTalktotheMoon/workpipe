# Software Engineer

**Company:** FanDuel
**Location:** Atlanta, GA
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
FanDuel is the premier mobile gaming company in the United States and Canada, seeking a Software Engineer to enhance their Sportsbook application. The role involves writing high-quality, maintainable code, improving architecture, and actively participating in team processes to deliver a best-in-class user experience.

## Core Skills
- React
- React Native
- JavaScript
- TypeScript
- HTML

## Minimum qualifications
- Degree in Computer Science, a related field, or equivalent practical experience
- 2+ years of software development experience building and maintaining mobile applications using React and React Native
- Strong proficiency in JavaScript/TypeScript and core web technologies (HTML and CSS)
- Experience delivering high-quality user interfaces with attention to performance, accessibility, and responsive design
- Experience working with Node.js services, ideally using a backend-for-frontend (BFF) pattern
- A solid understanding of data structures and algorithms
- Experience translating requirements into thoughtful design and delivering maintainable, testable code
- Familiarity with modern development workflows including Git/Github, pull requests, and code reviews
- Experience working with CI/CD pipelines, preferably Jenkins or BuildKite
- Experience working with cloud-based infrastructure, preferably AWS, including API Gateway and serverless patterns (e.g. Lambda)

## Preferred qualifications
- Experience with native mobile development on iOS and/or Android (e.g. Swift or Kotlin)
- Familiarity with OpenAPI or similar approaches to API documentation

## Responsibilities
- You will be working FanDuel Sportsbook application, providing our customers with a best-in-class experience
- You’ll play a pivotal part in making our platform highly usable and will write clear, scalable, and high-quality code
- You will work continuously to improve architecture and processes to improve quality and efficiency
- Your work is consistently of high quality
- You solve difficult problems, applying appropriate technologies and best practices
- You work with your team to invent, design and build software that is stable and performant
- You are proficient in a broad range of design approaches and know when it is appropriate to use them and when it is not
- Your solutions are pragmatic
- You consider the legacy of the code you produce and write code that an Engineer unfamiliar with the system can understand
- You limit the use of short-term workarounds
