# Engineer Systems

**Company:** Motorola Solutions
**Location:** West Valley City, UT
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Motorola Solutions is a global community dedicated to enhancing safety through technology. They are seeking an Engineer Systems to design and implement full-stack GIS applications, develop web service platforms, and lead software quality assurance efforts.

## Minimum qualifications
- Masters degree or Bachelors degree + 2 years experience

## Responsibilities
- Design and implement full-stack GIS applications, including frontend mapping interfaces and backend services that interact with CAD systems via custom API commands and data exchange protocols
- Complete development and testing to simulate and validate real-time GPS data within the GIS environment
- Develop and maintain web service platform integrating data processing for internal and external system communication
- Maintain and update mobile GIS application used across departments for field operations and on-site spatial validation to support real-time spatial data collection and validation
- Own and deliver key modules within the systems, including address validation services, map layer rendering logic, and geolocation workflows
- Lead root cause analysis (RCA) for production issues and system anomalies across GIS platforms while serving as the team’s point person for software quality assurance, including identifying long-term solutions and ensuring continuous improvement in code quality and system reliability
- Actively manage CI/CD pipelines across all GIS-related projects, integrating build automation, test execution, and deployment workflows
- Develop GUI-based automated testing to support regression coverage and catch visual interface issues during continuous integration cycles
- Contribute across the full software development lifecycle, requirements analysis, system design, coding, testing, debugging, and documentation, with a focus on spatial accuracy, maintainability, and platform integration
