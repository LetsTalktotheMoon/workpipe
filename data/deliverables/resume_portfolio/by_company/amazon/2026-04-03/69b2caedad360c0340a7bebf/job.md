# Software Development Engineer , Alexa Device Platform Team

**Company:** Amazon
**Location:** Austin, TX
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Amazon is a leading consumer electronics company, and they are seeking a Senior Embedded Software Engineer to join their Devices team. The role involves designing, architecting, and implementing innovative system solutions for Amazon Echo products.

## Core Skills
- Software Development
- Software Design
- Software Architecture
- Programming Languages
- System Performance Profiling

## Minimum qualifications
- 3+ years of non-internship professional software development experience
- 2+ years of non-internship design or architecture (design patterns, reliability and scaling) of new and existing systems experience
- Experience programming with at least one software programming language

## Preferred qualifications
- 3+ years of full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations experience
- Bachelor's degree in computer science or equivalent

## Responsibilities
- Investigate, prototype and deliver new and innovative system solutions
- Responsible for key domain areas like security, graphics, video and storage
- Work cross function with vendors
- OS, application teams and system architecture team for system resource and workloads for echo products
- Profile
- Analyze and Optimize system level performance for example kernel scheduler tuning, use/create profiling and trace tools to debug critical issues for Beta and production software
- Participate in design reviews
- API development, and documentation
- Demonstrate expertise in building AI bots and integrate AI tools into the standard workflow to improve team’s support load and test automation
