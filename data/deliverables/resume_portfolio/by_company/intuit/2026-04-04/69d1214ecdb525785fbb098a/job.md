# Software Engineer 2, AI Builder

**Company:** Intuit
**Location:** San Diego, CA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Intuit is a company on a mission to build disruptive products and enhance its ecosystem using AI technologies. The Software Engineer 2, AI Builder role involves developing full-stack software solutions and leveraging AI tools to innovate and experiment rapidly.

## Core Skills
- Full-Stack Software Engineering
- Typescript
- Python
- AI coding tools
- Startup Mentality

## Minimum qualifications
- Proven experience as a Full-Stack Software Engineer building modern, scalable applications across both the frontend and backend
- Typescript + Python is what we use
- Hands-on experience and deep comfort using AI coding tools (e.g.
- Claude Code
- Cursor
- Codex) to augment and accelerate your daily development workflows
- An aggressively entrepreneurial spirit with a track record of moving fast, cutting through red tape, and delivering results quickly
- A strong desire to be close to the customer, with demonstrated experience running A/B tests, prototypes, or rapid user experiments
- The ability to self-direct, pivot rapidly when hypotheses are disproven, and thrive in an unstructured, highly ambiguous environment
- A relentless appetite for learning new technologies, taking calculated risks, and having a blast while building the future

## Responsibilities
- Design, build, and ship full-stack software solutions with an intense focus on velocity and getting things done
- Extensively leverage AI coding assistants and modern tooling to write, test, and deploy code faster and more efficiently than traditional methods
- Work on the front lines with end-users to conduct rapid product experiments, gather real-world feedback, and iterate quickly based on data
- Operate comfortably in a fast-paced environment with a high degree of ambiguity, taking extreme ownership of your projects from inception to launch
- Build entirely new
- AI-native products from scratch while also identifying ways to accelerate existing Intuit products with cutting-edge tech
