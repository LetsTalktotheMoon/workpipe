# System & Cloud Engineer

**Company:** PayPal
**Location:** Scottsdale, AZ
**Seniority:** Entry Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
PayPal has been revolutionizing commerce globally for over 25 years, creating innovative experiences for consumers and businesses. They are seeking a Cloud Engineer to join their Cloud Platform Engineering team, responsible for building and managing PayPal's Public Cloud Infrastructure to ensure seamless application management and operation.

## Core Skills
- Public Cloud Technologies
- Google Cloud Platform (GCP)
- Amazon Web Services (AWS)
- GoLang
- Java

## Minimum qualifications
- 1+ years relevant experience and a Bachelor's degree OR Any equivalent combination of education and experience

## Preferred qualifications
- Experience working in Public Cloud Technologies and Tools which are required to build scalable solutions in Public Cloud Providers like GCP/AWS
- Experience with at least one programming language like GoLang
- Java
- Ruby
- Python to design/build/test public cloud platform backend and automation tools
- Experience with Infrastructure as Code (IaC) tools like Terraform
- Harness
- Ansible
- Knowledge of container orchestration tools (Kubernetes (GKE/EKS)
- Docker
- Puppet, etc.) to operate both Cloud-based and OnPrem production systems during our cloud migration journey
- Ensure infrastructure follows best security practices (IAM, encryption, secrets management)
- Experience triaging issues in Public Cloud environment and analyzing root causes
- Work cross-functionally across internal engineering teams, internal customers and product managers in various geo-locations across the world
- An eagerness to learn new technologies and partner with various teams giving PayPals internal Public Cloud users the best experience
- Experience setting up and maintaining monitoring and logging systems (e.g.
- DataDog
- Prometheus
- Grafana)
- Experience administering Production Linux systems in a 24x7 Production environment

## Responsibilities
- Implements tasks within the Software Development Lifecycle (SDLC), receiving structure and oversight from more experienced staff
- Follows well-established internal conventions and standard procedures
- Understands internal standards & processes and applies them to make technical decisions
- Collaborates with peers, manager, and project lead to gain understanding of tasks and review solutions
- May contribute to code & design reviews
