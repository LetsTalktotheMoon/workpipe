# Software Engineer II - Scala

**Company:** The Walt Disney Company
**Location:** New York, NY
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
The Walt Disney Company is a leading diversified international family entertainment and media enterprise. They are seeking a Software Engineer II to join the API Services team, focusing on building highly available and high throughput services that deliver quality content to subscribers.

## Core Skills
- Scala
- Java
- Software Development Fundamentals
- Software Architecture
- Web Technologies

## Minimum qualifications
- Bachelor's degree in Computer Science or related field, or equivalent training or work experience
- 3 years of experience in a technical field
- You work diligently with team members and project management to understand requirements to provide technical solutions and direction
- You are comfortable with ambiguity and being part of conversations where discordant views are present
- You have an intermediate understanding of software development fundamentals and architecture, and you aren't dogmatic about technology choices
- You have experience with Java
- Scala, and associated technologies
- You have experience building and maintaining highly automated systems that leverage tools such as Jenkins
- Cloud Formation
- Terraform
- Kubernetes, and Spinnaker

## Preferred qualifications
- You preferably are interested in learning about/have experience with web technologies and web system architecture
- You preferably have knowledge/experience with AWS products and services (EC2
- S3
- Lambda
- DynamoDB
- ElastiCache
- CloudFront, etc.) or other cloud providers

## Responsibilities
- Be a part of a team of big data and backend engineers to deliver high value projects
- Build components of large-scale data platform for real-time and batch processing, and own features of big data applications to fit evolving business needs
- Participate in building next generation service platform on cloud-based big data infrastructure for batch and streaming data applications, and continuously improve performance, scalability and availability
- Participate in sprint planning, backlog refinement, and retrospectives as a team
- Build automated tests to ensure application reliability
- Participate in code reviews
