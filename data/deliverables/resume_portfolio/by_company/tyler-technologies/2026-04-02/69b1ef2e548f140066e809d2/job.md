# Software Engineer (Go, Typescript & PHP) Enterprise Supervision

**Company:** Tyler Technologies
**Location:** Plano, TX
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Tyler Technologies is seeking a Software Engineer to join their Enterprise Supervision team, which supports a large-scale platform for public sector organizations across the United States. The role involves maintaining and modernizing an application stack primarily built with PHP, TypeScript, and MySQL, while contributing to the introduction of Go-based APIs and services.

## Core Skills
- PHP
- TypeScript
- MySQL
- RESTful APIs
- Docker

## Minimum qualifications
- Bachelor's degree in Computer Science
- Software Engineering, or equivalent professional experience
- 3+ years of professional software development experience working on web or SaaS applications
- Experience with PHP backend development
- Experience working with TypeScript or modern JavaScript in web applications
- Solid understanding of MySQL or other relational databases, including query optimization and schema design
- Experience building or maintaining RESTful APIs
- Ability to troubleshoot and improve existing production systems
- Familiarity with Docker-based development environments
- Experience working in Agile development teams
- Strong communication and problem-solving skills

## Preferred qualifications
- Experience developing backend services in GO
- Experience working with AWS cloud services, particularly serverless or event-driven components (API Gateway
- Lambda
- EventBridge
- SQS
- Step Functions)
- Experience with PHP frameworks such as Laravel or Symfony
- Familiarity with Infrastructure as Code tools such as AWS SAM
- CloudFormation, or Terraform
- Experience with observability and monitoring tools such as CloudWatch
- X-Ray, or Datadog
- Experience working on legacy codebases undergoing modernization

## Responsibilities
- Maintain and enhance a large-scale web application built with PHP
- TypeScript, and MySQL
- Maintain and enhance scalable, high-performance applications using AWS-native event-driven services, including API Gateway
- EventBridge
- Step Functions
- Lambda, and SQS
- Diagnose and resolve production issues, performance bottlenecks, and reliability concerns across the platform
- Implement new features and improvements while ensuring system stability and maintainability
- Contribute to the design, development, and support of new backend services written in GO as part of ongoing system modernization
- Participate in debugging, root cause analysis, and production support activities
- Collaborate with QA
- Product
- DevOps, and other engineering teams to deliver reliable software in an Agile environment
- Write clean, maintainable, and well-tested code, with strong attention to documentation and operational readiness
- Contribute to CI/CD pipelines and cloud infrastructure in AWS environments
- Support the gradual modernization of legacy components and reduction of technical debt
