# Software Engineer

**Company:** Lucid Services Group
**Location:** Indianapolis, IN
**Seniority:** New Grad, Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Lucid Services Group is seeking a motivated Junior Software Engineer with a strong foundation in SQL Server and Power BI. In this role, you will help design, develop, and maintain data-driven applications and reporting solutions that support business decision-making.

## Core Skills
- SQL Server
- T-SQL
- Stored procedures
- Queries
- Power BI

## Minimum qualifications
- Bachelor's degree in Computer Science
- Information Systems, or related field (or equivalent experience)
- 0–2 years of experience in software development or data-related roles
- Basic to intermediate proficiency in Microsoft SQL Server (T-SQL, queries, joins, stored procedures)
- Experience with Power BI (report creation, dashboards, data modeling)
- Understanding of relational databases and data structures
- Familiarity with version control systems (e.g.
- Git)
- Strong problem-solving skills and attention to detail

## Preferred qualifications
- Exposure to ETL tools or data pipelines
- Experience with .NET
- C#, or Python
- Understanding of data warehousing concepts
- Familiarity with Azure services (Azure SQL
- Power BI Service, etc.)
- Internship or project experience involving data analytics or reporting

## Responsibilities
- Develop, maintain, and optimize SQL Server databases, stored procedures, and queries
- Build and enhance Power BI dashboards and reports for business stakeholders
- Collaborate with senior engineers and analysts to design scalable data solutions
- Assist in data integration
- ETL processes, and performance tuning
- Troubleshoot and debug application and data-related issues
- Participate in code reviews and follow best practices for software development
- Document technical processes, data models, and reporting logic
