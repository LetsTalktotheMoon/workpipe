# Software Engineer

**Company:** CVS Health
**Location:** NY - New York
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
CVS Health is focused on building a world of health around every individual, aiming to simplify health care for communities. They are seeking an experienced Java developer to drive the design, development, and implementation of scalable microservices architecture for their applications, collaborating closely with various teams to ensure best practices are followed.

## Core Skills
- Java
- JDK 17, later
- Cloud development
- PL/pgSQL

## Minimum qualifications
- 3 years of progressive professional Java enterprise application development experience using JDK 17 or later
- Minimum 2+ years cloud development experience
- 1+ year's experience on PL/pgSQL, including stored procedures, functions, triggers, and views
- Experience for the configuration and troubleshooting Cloud services
- Experience deploying an application on Cloud using Kubernetes Engine, etc
- Experience working with: Docker
- Kubernetes
- Redis, etc
- Working experience in: Maven
- Git
- DevOps, etc
- 2+ years' experience developing microservices and messaging based applications, preferably using Kafka or Google Pub/Sub
- Experience with API Gateways
- Bachelor's degree in computer science or related field. Advanced technical degree and/or related training a plus

## Preferred qualifications
- 3+ years of Java
- Spring Framework and PostgreSQL development experience on cloud
- Google Cloud Platform experience, specifically with Istio
- GKE
- IAM etc
- PostgreSQL
- Experience using Spring Framework 5.0 or later and Spring Boot
- REST API development experience
- Experience working with large datasets using PostgreSQL in the GCP infrastructure
- Knowledge of PostgreSQL design principles, query optimization, index management, integrity checks, statistics, and isolation level
- Experience with monitoring, developing metrics dashboards and observability toolsets
- Experience with APIGEE

## Responsibilities
- Driving the design, development and implementation of scalable and efficient microservices architecture for applications
- Working closely with the development, architecture and operations team to ensure best practices in microservices design and development
