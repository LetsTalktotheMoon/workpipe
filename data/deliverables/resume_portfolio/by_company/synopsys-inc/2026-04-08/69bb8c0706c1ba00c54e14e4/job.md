# DevOps Engineer II

**Company:** Synopsys Inc
**Location:** Canonsburg, PA
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Synopsys Inc is a global leader in engineering simulation software, helping innovative companies solve complex design challenges. They are seeking a motivated DevOps Engineer II to maintain and enhance build management processes, ensuring reliable software releases and supporting continuous integration and delivery pipelines.

## Core Skills
- Git
- CI/CD
- Automation scripting
- PowerShell
- Python

## Minimum qualifications
- Bachelor's Degree in Engineering
- Computer Science, or a related field with 2 years of professional experience or a Master's Degree
- Experience with source code versioning tools such as Git along with CI/CD practices for building, testing, and release of applications
- Proficient in automation scripting languages (PowerShell
- Python
- Perl
- Bash)
- Knowledge of CMake
- Azure DevOps, and programming languages (C/C++
- C#)
- Understanding of Linux and/or Windows operating systems
- Experience working with compilers such as Microsoft Visual Studio
- GCC, or Intel
- Willingness to learn about modern HPC chipsets and scalable simulation environments
- Exposure and working knowledge of ARM64 architecture and supporting compilers
- Knowledge of security best practices and vulnerability remediation for source code and applications

## Responsibilities
- Maintaining, monitoring, documenting, and testing product builds and packaging to ensure quality releases
- Preparing, configuring, deploying, and maintaining build agents to support continuous integration and delivery pipelines
- Conducting system tests to assess the security, performance, and reliability of build systems and processes
- Collaborating with developers to identify code and testing issues using tools such as TotalView, and reporting findings promptly
- Working with IT to maintain DevOps infrastructure and manage necessary packages for compiling, linking, and testing software
- Adjusting builds and packaging in partnership with development teams to meet evolving requirements and project goals
- Participating in Azure DevOps trainings to enhance your skills and knowledge in modern DevOps practices
- Operating under direct supervision, with regular feedback and mentorship from experienced staff and the DevOps Manager
