**Company:** TetraMem - Accelerate The World
**Location:** San Jose, CA
**Seniority:** Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
TetraMem is focused on accelerating technological advancements, and they are seeking an ML Model Development Engineer to enhance machine learning capabilities. The role involves developing novel techniques for model compression, implementing machine learning models, and optimizing existing models for performance and efficiency.

## Core Skills
- Machine Learning
- Algorithm Development
- C++
- Python
- CI/CD

## Minimum qualifications
- Master's Degree in Computer Science
- Computer & Systems Engineering or related
- One year of experience in Job offered
- Algorithm Engineer or related
- Experience in Machine Learning
- Experience in Algorithm Development
- Experience in C++
- Experience in Python
- Experience in CI/CD

## Responsibilities
- Develop and pioneer novel techniques for model compression to reduce computational complexity and memory footprint maintaining high model performance
- Implement machine learning models, handle end-to-end development, from data preprocessing to model training and evaluation
- Improve existing machine learning models by exploring new algorithms, fine-tuning parameters, and integrating the research findings
- Prototype and iterate on different model designs, experimenting with various architectures and techniques to identify optimal solutions
- Implement efficient and optimized code to ensure fast and reliable model inference on our new hardware platform
