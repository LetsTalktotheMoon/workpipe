# Software Engineer (NJUS)

**Company:** NetJets
**Location:** Columbus, OH
**Seniority:** Entry, Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
NetJets, the global private aviation leader for more than 60 years, provides the pinnacle of private travel. They are seeking a Software Engineer to drive innovation in software and applications or data solutions, focusing on complex problem-solving and designing software solutions that enhance airline operations and customer experience.

## Core Skills
- Java
- C/C++
- JavaScript
- Python
- Cloud services - AWS

## Minimum qualifications
- Bachelor's in Computer Science or Computer Engineering
- 2-4 years of experience
- Strong programming skills in languages such as Java
- C/C++
- Javascript, python
- Excellent communication and collaboration skills

## Preferred qualifications
- Previous work experience as Software engineer with 2-4 years' experience
- Experience and knowledge of cloud services like AWS
- Experience in web development, mobile development, or software engineering projects as part of educational curriculum
- Knowledge and experience in Agile methodologies

## Responsibilities
- Drive implementation of software thru analysis and design activities for new business features or value-based enhancements to existing software
- Collaborate with various internal teams like architecture, infrastructure operations, cyber security, data, and others to implement architectures that encompass broad perspectives and requirements
- Automate build, test, and deploy code and build pipelines
- Build testing automation and associated processes (Unit test coverage, testing automation, pipeline automation, code scanning, etc.) into the software product
- Work in an agile team environment to iteratively delivery value
