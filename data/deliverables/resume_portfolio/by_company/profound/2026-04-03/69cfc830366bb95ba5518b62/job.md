# Software Engineer, Frontend

**Company:** Profound
**Location:** Buffalo-Niagara Area
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Profound is on a mission to help companies understand and control their AI presence. As a Frontend Engineer, you will play a key role in shaping high-performance, visually stunning React applications that power the future of AI-driven marketing experiences.

## Core Skills
- React
- Next.js
- TypeScript
- State management
- Performance optimization

## Minimum qualifications
- Deep expertise in React
- Next.js, and TypeScript, with strong knowledge of state management and performance optimization
- Proven experience architecting and scaling complex Next.js applications (App Router
- SSR
- SSG
- ISR)
- Strong command of Tailwind CSS
- SASS, or SCSS, delivering polished, pixel-perfect UIs
- Experience with data-fetching libraries like React Query (or similar)
- Skilled in building smooth, engaging interfaces with advanced CSS animations, transitions, and layout techniques
- Strong understanding of caching strategies and CDN integrations for global performance at scale
- Experience with React Server Components in production environments
- Familiar with internationalization and localization strategies in Next.js applications
- Comfortable operating in a fast-paced startup environment with rapid iteration cycles
- A strong design eye and portfolio demonstrating high-quality front-end work

## Preferred qualifications
- Experience with GSAP for advanced animations
- Experience with Three.js
- React Three Fiber (R3F), or immersive 3D experiences

## Responsibilities
- Build and refine sleek, high-quality user interfaces from low-fidelity wireframes
- Implement god-like CSS to create exceptional, pixel-perfect experiences
- Optimize front-end performance to ensure seamless interactions
- Work closely with design and engineering to bring ideas to life with precision and speed
- Contribute to our product’s core architecture and front-end best practices
