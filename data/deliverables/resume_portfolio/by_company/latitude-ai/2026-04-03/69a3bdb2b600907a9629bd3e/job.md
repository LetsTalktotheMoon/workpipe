# Senior Software Engineer - Cloud Platform

**Company:** Latitude AI
**Location:** Palo Alto, CA
**Seniority:** Mid, Senior Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Latitude AI is a subsidiary of Ford Motor Company focused on developing automated driving technologies. The Senior Software Engineer for the Cloud Platform will architect and scale a hybrid-cloud Kubernetes platform, optimize workloads for machine learning, and provide technical leadership to ensure the success of engineering teams.

## Core Skills
- Python
- Go
- Java
- Infrastructure as Code
- Terraform

## Minimum qualifications
- Bachelor's degree in Computer Engineering
- Computer Science
- Electrical Engineering
- Robotics or a related field and 4+ years of relevant experience (or Master's degree and 2+ years of relevant experience, or PhD)
- Hands on development in Python
- Go, or Java to create production level services that incorporate CI/CD
- Experience in using infrastructure as code to automate the creation of cloud resources. E.g. Terraform
- CloudFormation, or others
- Strong experience with architecting/deploying scalable services and workloads in the cloud (AWS
- GCP) and/or cloud native environments
- Strong experience with deployment management/scheduling/high availability/failure modes for distributed applications and compute workloads running on CPUs and GPUs
- Understand engineering design limitations and provide guidance for teams to scale their deployments in the cloud so that they can achieve desired performance within budget
- Experience creating reusable deployment patterns for teams to adopt, that incorporate networking, security, monitoring, logging, alerting, continuous integration and deployments
- Understanding of API specification languages (e.g. Open API/gRPC) and the ability to create reusable code via code generation, templates, and other rapid development techniques for teams to consume
- Leverage and modify open source tools to meet unique requirements for internal teams, including but not limited to Kubernetes
- Prometheus
- FluentD
- Istio
- Envoy
- CoreDNS, etcd
- Flux, among others

## Responsibilities
- Architect and scale a multi-tenant, hybrid-cloud Kubernetes platform to ensure seamless workload portability across public and internal environments
- Develop custom Kubernetes operators and self-service abstractions to simplify complex infrastructure management for autonomous vehicle engineering teams
- Optimize high-performance distributed workloads by architecting specialized compute environments for ML and simulation on heterogeneous hardware
- Standardize end-to-end GitOps workflows and CI/CD pipelines to automate security, networking, and observability across the infrastructure lifecycle
- Provide technical leadership and architectural guidance to ensure engineering teams build resilient, cost-effective services aligned with cloud-native best practices
- Manage the lifecycle of critical platform components like service mesh and networking while proactively resolving systemic bottlenecks in hybrid cloud environments
