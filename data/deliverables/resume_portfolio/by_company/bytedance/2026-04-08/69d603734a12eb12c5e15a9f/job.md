# Backend Software Engineer - Security Data

**Company:** ByteDance
**Location:** San Jose, CA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
ByteDance is a company that inspires creativity and enriches life through innovative products. The Backend Software Engineer - Security Data will be responsible for designing and maintaining security data products and platforms that enable risk detection and security analytics while collaborating with cross-functional teams to translate requirements into actionable data solutions.

## Core Skills
- Backend Programming Languages
- Software Development Life Cycle (SDLC)
- Data Storage Systems
- Messaging Systems
- Data Processing Systems

## Minimum qualifications
- Bachelor's degree or above in Computer Science
- Software Engineering
- Data Engineering, or a related field, with 3+ years of experience building, operating, and scaling production backend systems
- Proficiency in at least one backend programming language (e.g.
- Go or Java), and familiarity with backend and data system development processes (SDLC)
- Hands-on experience with data storage, messaging, and processing systems, such as MySQL
- Redis
- Kafka, and Hive, and a solid understanding of distributed systems
- Experience designing and implementing data pipelines and data models for large-scale data processing scenarios
- Strong product awareness and a data-driven mindset, with experience turning security data into actionable product features or decision-making capabilities

## Preferred qualifications
- Solid experience designing backend systems, including RESTful APIs, microservices or service-oriented architectures, and data models using SQL and NoSQL databases
- Strong ability to collaborate cross-functionally with product, infrastructure, and security teams to deliver end-to-end backend solutions
- Experience with Big Data technologies (Hadoop
- Hive
- Spark
- Presto
- Flume
- Kafka
- ClickHouse
- Flink etc.)
- Experience in security-related data products (e.g., risk control, anti-fraud, security monitoring) is a plus
- Experience with real-time data processing, big data analytics, or security analytics platforms is a plus

## Responsibilities
- Design, develop, and maintain security data products and data platforms, enabling risk detection, security analytics, and data-driven security capabilities
- Collaborate closely with product managers, security teams, and business stakeholders to translate security and business requirements into data models, data pipelines, and productized security capabilities
- Build and optimize large-scale batch and real-time data processing pipelines to support security scenarios such as risk control, anomaly detection, and behavior analysis
- Ensure data quality, stability, and observability across the full data lifecycle, and continuously improve system architecture, performance, and engineering efficiency
- Develop a deep understanding of security scenarios and business context, and leverage data to drive measurable improvements in security effectiveness
