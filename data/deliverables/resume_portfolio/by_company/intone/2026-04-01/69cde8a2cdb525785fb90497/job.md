# Software Engineer (L2, Java, SQL)

**Company:** Intone
**Location:** Merrimack, NH
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Intone Networks is seeking a Software Engineer to join their data-focused team, responsible for building and maintaining cloud-based applications. The role involves designing and developing secure, scalable microservices and creating REST APIs while working with SQL databases and supporting deployments through continuous integration pipelines.

## Core Skills
- Java
- Spring Boot
- SQL
- PostgreSQL
- Snowflake

## Minimum qualifications
- 3 years minimum experience
- Bachelor's degree in computer science or a related field
- Strong skills in Java (especially Spring Boot)
- Strong skills in SQL
- Experience with database technologies such as PostgreSQL or Snowflake
- Experience in CI/CD tools like Jenkins
- Experience in test automation using JUnit
- Experience with messaging platforms like Kafka or SQS/SNS
- Familiarity with AWS architecture
- Familiarity with containerization

## Preferred qualifications
- AWS (SNS
- SQS
- IAM
- API Gateway, s3
- RDS – Postgres
- MSK)
- Kafka

## Responsibilities
- This role is for a Software Engineer working on a data-focused team, building and maintaining cloud-based applications
- The position involves designing and developing secure, scalable microservices using modern Java frameworks, as well as creating REST APIs and handling communication between services through messaging systems
- Responsibilities also include working with SQL databases, writing automated tests, packaging applications in containers, and supporting deployments through continuous integration pipelines in AWS environments
