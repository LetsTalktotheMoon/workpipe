# Software Engineer II

**Company:** Waystar
**Location:** Atlanta, GA
**Seniority:** Entry, Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Waystar is a company that simplifies healthcare payments through a smart platform and better experience. They are seeking a Software Engineer II to join their team, responsible for building scalable software frameworks, supporting web applications, and mentoring junior engineers.

## Core Skills
- Full stack web development
- Object-oriented programming
- PHP
- Relational databases
- MySQL

## Minimum qualifications
- 2+ years of professional full stack web development experience
- Object-oriented programming (we're mostly PHP)
- Relational databases (we use MySql)
- Modern JavaScript frameworks such as Angular
- React, or Vue (we use Vue)
- JavaScript
- HTML
- CSS, and/or MVC Frameworks
- Unit and integration testing methodologies
- Distributed system architecture and asynchronous processing using queues (AWS SQS)
- Automated production logging, monitoring and alerting
- Agile development

## Preferred qualifications
- Bachelor's Degree in Computer Science or related field preferred (or equivalent experience)

## Responsibilities
- You will join a team of full stack engineers to build new features for our web applications, as well as support and maintain them
- You should expect to move up and down the software stack, writing client-side JavaScript
- HTML, and CSS, server-side PHP and the SQL statements for the products you release
- Collaborate with Product
- UX and other engineering teams to solve difficult problems and design new features
- Scope out technical designs that enable your team to execute on proposed solutions
- Upgrade legacy apps and code to modern frameworks and technologies
- Mentor junior engineers and promote a culture of learning and teaching
- Embrace an agile development process that emphasizes test automation, code reviews, and fast release cycles
- Be a communicator! Have opinions and the ability to justify them - we want to know what you think
