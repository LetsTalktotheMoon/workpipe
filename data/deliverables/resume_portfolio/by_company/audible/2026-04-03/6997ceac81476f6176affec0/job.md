# Quality Assurance Engineer II (ADBL185)

**Company:** Audible
**Location:** Cambridge, MA
**Seniority:** Mid, Senior Level

## About the job
Audible, Inc. is a leading company in audio entertainment, and they are seeking a Quality Assurance Engineer II. The role involves leading the testing of large-scale software systems and developing automated test cases to ensure high-quality software delivery.

## Core Skills
- Manual testing
- Automated testing
- Test strategy development
- Test automation for front end applications
- Full stack testing

## Minimum qualifications
- Requires a Master's degree in Computer Science
- Engineering
- Mathematics, or a related field plus two (2) years of experience as a Quality Assurance Engineer
- Software Test Developer, or a related occupation
- Experience may be gained concurrently and must include:
- Two (2) Years In Each Of The Following:
- Documenting and coordinating the deployment of manual and automated tests for complex systems
- Owning full test strategy through test completion for major components of a software system
- Test automation for consumer-facing front end applications
- Testing across a full stack system, including Front End
- Services, or databases
- Writing Java or Python based test automation
- Testing in an Agile development environment

## Responsibilities
- Lead the testing of large-scale software systems
- Create test plans, test cases, and drive continuous improvements to the quality assurance processes using AWS
- Internet and Web technologies including HTTP
- XML
- HTML
- Load Balancing, and DNS
- Develop software tests using Java
- Python
- AWS Lambda, and other tools, document and coordinate the deployment of manual and automated tests for complex systems, and test mobile device development kits and device frameworks
- Design and build automated test cases for mobile device-based graphical user interfaces using test automation frameworks such as Selenium and Java
- Design, develop, and execute quality assurance test cases and support tests for large-scale, distributed software systems using Java including white and black box tests
- Work closely with engineers, business owners, and customer service professionals throughout the software development and testing lifecycle, including the requirements gathering and design
