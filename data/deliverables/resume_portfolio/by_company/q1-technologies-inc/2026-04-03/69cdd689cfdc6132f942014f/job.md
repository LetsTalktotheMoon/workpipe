# Data Scientist with ML experience

**Company:** Q1 Technologies, Inc.
**Location:** Midvale, UT
**Seniority:** Mid Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Q1 Technologies, Inc. is seeking a Data Scientist with ML experience to develop, validate, and deploy machine learning models for real-world business use cases. The role involves data cleaning, exploratory data analysis, and collaboration with cross-functional teams.

## Core Skills
- Python
- Machine Learning algorithms
- Deep Learning frameworks
- SQL
- Data Visualization

## Minimum qualifications
- Strong proficiency in Python (NumPy
- Pandas
- Scikit-learn, etc.)
- Experience with ML algorithms (regression, classification, clustering
- NLP, etc.)
- Hands-on experience with deep learning frameworks (TensorFlow / PyTorch – preferred)
- Knowledge of SQL and working with large-scale datasets
- Experience in data visualization tools (Power BI
- Tableau
- Matplotlib
- Seaborn)
- Understanding of statistics, probability, and data modeling
- Familiarity with cloud platforms (AWS / Azure / GCP) is a plus
- Experience with MLOps and model deployment is an advantage

## Preferred qualifications
- Bachelor's/Master's degree in Computer Science
- Data Science
- Statistics, or related field
- Experience working in consulting or client-facing roles
- Strong problem-solving and communication skills

## Responsibilities
- Develop, validate, and deploy machine learning models for real-world business use cases
- Perform data cleaning, preprocessing, and feature engineering on large datasets
- Conduct exploratory data analysis (EDA) to uncover trends and insights
- Collaborate with cross-functional teams including business stakeholders, data engineers, and analysts
- Implement and optimize algorithms for performance and scalability
- Work on end-to-end model lifecycle: development, testing, deployment, and monitoring
- Communicate findings and insights through visualizations and reports
- Stay updated with the latest advancements in data science and ML technologies
