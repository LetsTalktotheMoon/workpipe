# Delivery Platform Software Engineer 1 - Developer Tools Team (Hybrid - Seattle, WA)

**Company:** Nordstrom
**Location:** Seattle, WA
**Seniority:** Entry Level
**Work Model:** Hybrid
**Employment Type:** Full-time

## About the job
Nordstrom is revolutionizing its technical landscape with an integrated ecosystem of platforms and tools. The Delivery Platform Software Engineer will take full ownership of the development lifecycle of DevOps platform services, enhancing developer productivity and experience.

## Core Skills
- Python
- Golang
- Java
- JavaScript
- TypeScript

## Minimum qualifications
- Bachelor's or Master's Degree in Computer Science or related field
- At least 1 year of industry experience with one or more languages (Python
- Golang
- Java
- JavaScript/TypeScript)
- Experience with CI/CD pipeline development on GitHub Actions or similar platforms (GitLab
- Jenkins)
- Experience building backend APIs or services (REST, gRPC, or similar)
- Excellent problem-solving skills with a proactive approach to identifying and resolving issues
- Strong communication skills and the ability to work collaboratively with cross-functional teams
- Familiarity with containerization technologies (Docker) and orchestration platforms
- Basic understanding of cloud providers (AWS
- GCP, or Azure)
- Eagerness to learn and adopt AI-assisted development tools

## Preferred qualifications
- Experience with GitHub platform features (Actions
- Apps
- API, webhooks)
- Hands-on experience with AI coding assistants (GitHub Copilot
- Claude, or similar)
- Familiarity with Kubernetes and container orchestration
- Experience with Infrastructure as Code (IaC) tools like Terraform
- CloudFormation, or Kubernetes custom resources
- Understanding of CI/CD best practices, including security scanning, artifact management, and deployment strategies
- Experience with observability and monitoring tools (NewRelic
- DataDog
- Prometheus)
- Background in platform engineering or developer experience optimization
- Experience with workflow orchestration tools (Temporal
- Airflow, or similar)
- Familiarity with data analytics
- ETL pipelines, or working with large-scale datasets
- Experience building AI agents or integrations with AI APIs (OpenAI
- Anthropic, or similar)

## Responsibilities
- Design, build, and maintain developer tools and platforms supporting our GitHub-based DevOps Platform
- Collaborate with cross-functional teams to define and develop innovative solutions that enhance developer productivity
- Build and optimize CI/CD pipelines using GitHub Actions, integrating with platforms such as Artifactory
- Vault
- Certificate Management, and public cloud services
- Develop and manage automated deployment processes and GitHub workflows
- Build backend services and APIs that power platform capabilities, including policy enforcement and deployment workflow logic
- Design and implement workflow orchestration using Temporal to automate and govern deployment and policy processes
- Develop AI agent integrations and MCP (Model Context Protocol) server capabilities to extend AI-assisted developer tooling
- Enable and support AI-assisted development workflows using GitHub Copilot and Claude to enhance developer experience
- Facilitate communication and collaboration among development, operations, and other teams to improve efficiency
- Monitor and troubleshoot platform services to ensure smooth operation and minimal downtime
