# Asset & Wealth Management - Software Engineer - Associate - New York

**Company:** Goldman Sachs
**Location:** New York, NY
**Seniority:** New Grad, Entry Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Goldman Sachs is one of the world's leading asset managers, dedicated to helping clients achieve their investment goals. The role involves architecting and building scalable platforms for investment data, leveraging modern technologies to create resilient services in a real-time trading environment.

## Core Skills
- Microservices
- Java
- React
- SQL
- Non-relational databases

## Minimum qualifications
- Meaningful experience in, but not limited to, any one of the following: Microservices
- Java
- React
- SQL
- Experience of working with non-relational as well as relational databases
- Strong knowledge of data structures and algorithms
- Excellent object oriented or functional analysis and design skills
- Comfortable multi-tasking, managing multiple stakeholders, and working as part of a global team
- Proven communication and interpersonal ability

## Preferred qualifications
- Experience building client- and consumer-facing products is a plus, but far from required
- Knowledge of existing strategic firmwide platforms is a plus, but far from required

## Responsibilities
- Join the Product Reference Data Technology group to lead the evolution of our Asset Management data ecosystem
- Architect and build robust, scalable platforms that serve as the 'golden source' for investment decisions
- Leverage a cutting-edge stack—including Java 17
- Vert.x 3.x
- Kafka, and SingleStore—to deliver low-latency, highly resilient services capable of supporting a real-time trading environment
- Be part of a global team, working closely with portfolio managers, operations, and other engineers
- Build low latency, scalable and resilient services
- Write high quality, well-tested, and scalable code
- Evaluate the short- and long-term implications of every implementation decision
- Grow professionally and learn from other accomplished software engineers through pair coding while helping your peers to do the same
- Collaborate on critical system architecture decisions
- Review and provide feedback on other developers’ code and design
