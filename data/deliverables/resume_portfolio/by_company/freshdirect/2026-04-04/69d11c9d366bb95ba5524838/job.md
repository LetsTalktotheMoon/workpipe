# Mobile React Native Developer

**Company:** FreshDirect
**Location:** Woodbridge, NJ
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
FreshDirect is a technology-powered grocery company focused on delivering the freshest food directly from farms to homes. They are seeking a Mobile React Native Developer to contribute to the development of mobile applications, working closely with product and design teams to enhance user experience and integrate with backend services.

## Core Skills
- React Native
- JavaScript
- React Native IDEs
- React Native tools
- React Native UI components

## Minimum qualifications
- 3+ years of React Native Development
- Hands-on code contribution and delivery
- Working with the product and design teams to understand end-user requirements and user stories, then translating that into an effective technical approach
- Developing and testing your mobile application code, leveraging a sprint based
- Agile methodology
- Familiarity with APIs and integrate mobile applications with existing backend services
- Working closely with QA members on the team to ensure the code you develop meets acceptance criteria, including doing rework where necessary
- Coordinate and work closely with other distributed teams across multiple time zones, to execute solutions
- Participating in brainstorming sessions and having the chance to contribute innovative and original ideas to our technology and products
- Proven track record of designing and developing visually appealing mobile applications for iOS and/or Android devices using React Native
- Excellent JavaScript skills with an in-depth knowledge of React Native IDEs
- React Native tools
- React Native UI components and React Native testing tools
