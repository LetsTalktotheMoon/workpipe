# Software Dev Engineer II (SDE2), Amazon Connect

**Company:** Amazon Web Services (AWS)
**Location:** Seattle, WA
**Seniority:** Mid Level
**Work Model:** Onsite
**Employment Type:** Full-time

## About the job
Amazon Web Services (AWS) is a leading cloud platform that provides business applications to millions of companies worldwide. They are seeking a Software Dev Engineer II to innovate and develop the next generation of Amazon Connect, a cloud-based contact center that leverages AI and AWS services to enhance customer service experiences.

## Core Skills
- Software Development
- Software Design
- Software Architecture
- Design Patterns
- System Reliability

## Minimum qualifications
- 3+ years of non-internship professional software development experience
- 2+ years of non-internship design or architecture (design patterns, reliability and scaling) of new and existing systems experience
- Experience programming with at least one software programming language

## Preferred qualifications
- 3+ years of full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations experience
- Bachelor's degree in computer science or equivalent

## Responsibilities
- Own code end-to-end from design to development
- Work closely with senior technical leaders within the team and across AWS
- Develop scalable, resilient systems while maintaining a constant customer focus
